# Normalization Scoping Tests

These tests support Track B prediction extraction by checking whether built GU structure fixes
normalizations strongly enough to emit a native numerical prediction.

| script | purpose |
|---|---|
| `pred_norm_rank.py` | exact monomial-rank receipt for `PRED-NORM-RANK`: under the current allowed field, DeWitt-scale, and source-normalization rescalings, only scale-relative combinations survive; no native value is fixed for `mu_DW`, `Z_U*kappa`, or `source_norm` |

The scoping tests are not claim-status machinery. They should not move canon, verdicts, public posture, or
paper status by themselves.
