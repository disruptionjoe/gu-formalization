# published-papers/

This folder is **empty by default**. It holds only papers that have been **published publicly**
(arXiv, a journal, a blog post that is live, etc.). Joe moves a file here by hand when, and only when,
it actually goes public.

Everything still in progress lives in `../draft-papers/`. The two-folder split is deliberate:

- `draft-papers/` — work in progress, drafts, notes, white papers, the prepublish review tracker. Nothing
  here is a public claim. Edit freely.
- `published-papers/` — the public record. A file here corresponds to something with a real external
  home. Treat as append-mostly; do not silently rewrite a published artifact (publish a revision instead,
  and note it).

When a draft is published, record alongside it (or in this README) the venue, the date, and the canonical
external URL / DOI / arXiv id.
