## Scales

Novelty:

- 1 = already standard
- 2 = standard but newly useful here
- 3 = adjacent or implicit in literature
- 4 = likely original synthesis from existing ingredients
- 5 = genuinely unexplored research direction

Profundity:

- 1 = bookkeeping or framing convenience
- 2 = clarifies the lane
- 3 = shifts research prioritization
- 4 = opens a substantial research program
- 5 = changes the level at which the problem is understood

Testability:

- High = an agent can run a meaningful local test now
- Medium = agent can build a rigorous scoping artifact or toy model, but proof needs expertise
- Low = currently too broad, likely needs outside collaborators

Publish potential:

- 1 = park internally
- 2 = useful internal note or appendix only
- 3 = public issue / contributor prompt after cleanup
- 4 = repo-worthy synthesis or strong public open-problem artifact
- 5 = paper-grade or public lead claim if written with proper scope

## Executive Reorientation

V1 asked which paths were novel and profound. V2 asks a sharper question:

> Which paths survive first contact with formal pressure, and what posture should they have in public?

The answer is:

1. Public-facing repo should lead with bounded specification and falsification artifacts, not a claimed GU proof.
2. The strongest mathematical spine is now signed-readout/provenance factorization, not raw CALM/GW equivalence.
3. The strongest contributor surface is still the no-go map, six-axis protocol, Type II1 checklist, and bounded theorem-core tasks.
4. C_MPR is useful, but should be introduced as a proposed formalization program.
5. The Hegelian/persona method is valuable, but only if every synthesis gets a post-synthesis layer audit.

- `proved`

1. no-go assumptions and evasions,
2. six-axis specification discipline,
3. Type II1 requirements,
4. signed-readout theorem core,
5. worked examples and counterexamples.

Keep the profound but underproved architecture visible, but label it honestly:

> C_MPR is a proposed formalization program for observer-accessible monotone shadows of signed/global provenance substrates.

That sentence preserves the metalayer insight without pretending the theorem has already been written.
