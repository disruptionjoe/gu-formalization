# One-time triage pass (2026-06-30)

This is a rating pass, not a proof pass. Each lane was scored on **Novelty / Publishability / Profundity (0-5)**, given a **decisive first test**, and assigned **PURSUE / PARK / KILL**.

## Calibration note (read first)

Scores were re-normalized to one absolute rubric:

- **5** = new structure that reframes a field / top-journal standalone
- **4** = solid new GU-independent theorem, real-journal grade
- **3** = referee-checkable note or lemma with genuine new content (clean arXiv-note grade)
- **2** = expository / negative / dictionary note; restates known facts in new framing
- **1** = appendix lemma / internal honesty fix; textbook content applied to the repo
- **0** = internal process bookkeeping, no external value

**The single most important fact: nothing in this set scores above 3 on any axis.** The whole program is reconstruction/note-grade. A "3" here is a clean arXiv note, not a result that moves a field. Composite = N + P + Prof; it measures *promise of content*, not *remaining work*.

`src` = the lane's decisive gate routes through the unbuilt RS/IG **source action** (Y / N / partial).
