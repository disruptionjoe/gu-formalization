# Pre-registration: the SOURCE-DOMAIN-SELECTOR swing

## Why this and not "prove the pencil" or "enumerate the lattice"

Scoping the pencil theorem surfaced that its only open piece and B5's block are
the **same missing object**, and it is also the L7 gap the C1 run just
re-flagged:

| face | item | what it needs |
| --- | --- | --- |
| operator theorem | `OPERATOR-END-PENCIL` | a source-owned endpoint / noncompact-asymptotic relation, fixed independently of `z` and `δ`, to select the deck-compatible Krein-self-adjoint domain |
| source action | `B5-MIDDLE-DIFFERENTIAL` | the source-action's typed middle contraction + **domain/boundary data** + real/Krein closure |
| construction lattice | `CONSTRUCTION-SPACE` L7 | the `(9,5)` Krein form's physical superselection + probability rule |

All three bottom out in **the source-owned boundary/domain data of the actual
GU construction**, which is currently only partially extracted. On a *fixed*
domain the pencil theorem is already proved (`COMPACT-GRAPH-CONDITIONAL`,
`COMPACT-PENCIL-REGULAR`); re-swinging at "prove the pencil" only re-hits
`DOMAIN-OBSTRUCTION-SOURCE`. The lattice enumeration was just shown
non-generative (C1: 1 of 8 slots, 87 open). So the highest-leverage Lane-1
work is neither — it is the **shared source-domain selector**. Resolving it
either way unblocks two papers at once and pins the L7 gap.

New leverage that did not exist when these items were parked: the 2026-07-20
ingest of fresh Weinstein primary material (Portal GU + DarkHorse transcripts,
`lab/sources/`) has **not** been mined against these *specific* endpoint /
domain / boundary demands. That is the tractable opening.

## The two-pronged swing (pre-declared)

### Prong A — SOURCE-EXTRACTION (run first; cheaper, higher readiness)
Mine the full Weinstein primary corpus (drafts + Portal + DarkHorse transcripts
+ the existing `claim-mining-toe-weinstein-2026-07-20.md`) against ONE frozen
demand list: does the primary source anywhere specify, at construction grade —
1. an **endpoint / boundary relation** for the first-order operator (a
   `T^*H(b)T = H(a)`-type deck-compatible isotropic condition, or an explicit
   endpoint phase), or
2. a **noncompact asymptotic domain** condition (decay / self-adjoint domain at
   the fiber ends), independent of `z` and `δ`, and
3. the B5 middle-contraction typed matrix + signs that the same boundary data
   would co-determine.

Output: a typed extraction ledger, each row = {demand, source-locus or
`SOURCE-SILENT`, construction-grade or comparator-grade}. This is a
native-vs-imported typing exercise; imports are counted, never hidden.

### Prong B — OBSTRUCTION PROOF (run only if Prong A returns SOURCE-SILENT)
Attempt the honest no-go: prove there is **no** deck-compatible maximal
isotropic domain that current sources can select — i.e. a boundary-monodromy
obstruction on every deck-compatible maximal isotropic domain (the item's own
`kill_condition`). If it closes, the shared claim of both papers is honestly
demoted to *conditional*, and the **exact external datum GU must import** is
named. A pinned "here is precisely the one missing source input" is a real,
publishable localization, not a null.

## Pre-declared outcomes (kill-conditions BEFORE computation)

- **S-SELECT (win):** Prong A finds a construction-grade source-owned endpoint/
  asymptotic relation → feed it to `OPERATOR-END-PENCIL` (closes the theorem on
  the true noncompact fiber, unblocks both papers) AND to `B5-MIDDLE-DIFFERENTIAL`
  (co-determines the middle contraction). Quarantine for hostile verify.
- **S-CONDITIONAL:** Prong A finds only comparator-grade / imported boundary
  data → both papers stay conditional; the import is typed and counted; state
  the exact native gap. This is the honest current ceiling, now sharpened.
- **S-OBSTRUCTED:** Prong B closes the no-go → shared claim demoted to
  conditional-forever-without-import; the exact missing datum is named. A
  finding, not a failure.
- **S-SILENT-OPEN:** Prong A is SOURCE-SILENT and Prong B does not close →
  `DOMAIN-OBSTRUCTION-SOURCE` confirmed and localized; route the missing-datum
  demand to P2C (native-vs-forced owner) as a frozen packet, park with the exact
  spec of what would reopen it.

## Controls (mandatory — the discipline that earns the verdict)

- **Planted positive:** insert a fabricated "source specifies endpoint phase φ"
  row; the extraction contract MUST flag it as unsourced or the extraction has
  no discriminating power (same failure mode as the fixed-δ uniformity control
  and the weak-value over-claim — a control that cannot fail proves nothing).
- **Native-vs-import typing:** every extracted datum tagged construction-grade
  vs comparator/imported; a conventional-physics completion (BRST, positive-
  Hilbert, APS, standalone-RS) may be priced as a comparator but never
  substituted for the missing native datum.
- **Hostile verify:** any S-SELECT is quarantined and adversarially checked
  (does the extracted relation actually fix the domain independent of `z,δ`, or
  is it a `z`-dependent surrogate?) before it moves either item's state.

## Forbidden shortcuts (carried from both parent items)
