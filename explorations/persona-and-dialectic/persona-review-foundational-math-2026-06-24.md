## Persona 01: Differential Geometer

### Missing or underweighted now

The repo has accepted the metric-bundle reading of `Y^14 = Met(X^4)`, but it is still
underweighting typed geometric governance: one authoritative definition of the rolled-up
operator, the horizontal-normalized second fundamental form, the gimmel metric, the section
variation, and which geometric quantities are fields versus background data.

The VZ convention note shows the risk clearly. `Phi_2`, `Phi_2 o d_A`, `Phi_2(F_A tensor -)`,
and `F_xi` are all compatible if separated, but the repo still needs a primary typed operator
definition before the VZ E-block can become proof-grade.

### Steelman

Write a "typed geometry spine" for `Y^14`: define `Met(X)`, `g_gimmel`, the horizontal split,
`II_s^H`, the rolled-up `D_roll`, all `Phi` variants, and the reduction dictionary from
fields on `Y` to section-pulled 4D fields. This is not cleanup. It is the geometric language
without which the action, VZ, RS, and dark-energy gates keep talking past one another.

### Hegelian path against the steelman

Thesis: a canonical metric-bundle geometry should be able to host all later gates once the
objects are typed.

Antithesis: the current notes show that the same symbols can refer to different operator
orders or variation statuses. That ambiguity can manufacture false closure, especially for
VZ and theta dynamics.

Synthesis / aufhebung: preserve the metric-bundle insight, but relocate "GU works" from a
dimension count to a typed-operator contract. The geometry is allowed to lead only after it
states the exact maps and variational roles used downstream.

Closure / falsification condition: close this gate when a single canonical file defines the
typed `Y^14` operator/action conventions and all current VZ/action notes can cite it without
renaming. Falsify the steelman if the necessary conventions cannot be made mutually coherent.

### Priority verdict

lead

## Persona 02: Gauge Theorist

### Missing or underweighted now

The repo is still underweighting gauge selection. It has strong local data: Sp(64), shiab,
Pati-Salam branching, Type II_1 hosting of finite CC data, and anomaly checks at partial
grade. But it has not derived the SM compact gauge group, hypercharge normalization, or
finite Connes algebra from GU or Type II_1 structure. The Type II_1 selector work makes this
especially explicit: hosting `A_F` is not selecting `A_F`.

### Steelman

Build a genuine gauge-shadow selector: a functor from GU/Type II_1 data to finite
Connes-Chamseddine control data that recovers `A_F = C + H + M_3(C)`, the compact SM gauge
group, hypercharges, and one-generation representation content without importing them as
inputs. A narrower acceptable win would be a theorem that Type II_1 data canonically selects
three equal CC-generation sectors while `A_F` is supplied separately.

### Hegelian path against the steelman

Thesis: gauge theory demands exact selection of group, representations, anomalies, and
couplings, not just a large ambient group.

Antithesis: the current strongest construction embeds known finite CC data into a Type II_1
host; C3/D4 gives only a count-label toy; the anti-smuggling theorem filters out any selector
whose result survives the replacement `C3 -> Cn`.

Synthesis / aufhebung: keep the Type II_1 lane as a host and possible selector lab, but make
"selection beyond embedded CC" the gauge-theoretic pass/fail condition.

Closure / falsification condition: pass if a non-smuggling selector recovers at least one
SM datum not manually embedded. Fail if every working construction must first supply `A_F`,
the one-generation module, and three copies.

### Priority verdict

protect

## Persona 03: Algebraic Topologist

### Missing or underweighted now

The repo now knows that raw RS ranks are not the generation count. The missing object is the
K-theory symbol class of the constrained/gauge-fixed RS operator on K3, including Chern
character terms, quaternionic structure, ellipticity, and ghost or gauge-fixing data.

The algebraic topologist would also flag the continuing temptation to turn `Ahat(K3)=2`
plus a target number into a derivation. The June 24 rank gate explicitly rejects that.

### Steelman

Compute the RS constrained/gauge-fixed symbol class and Atiyah-Singer/APS index on K3 from
first principles. The deliverable should construct the bundles, projectors, gauge complex,
right-H conversion, Chern character, and final `ind_H(D_RS)` without using 8, 4, 24, or three
generations as inputs.

### Hegelian path against the steelman

Thesis: the generation question is now a topological-index question, not a numerology or raw
rank question.

Antithesis: the explicit Clifford/projector computation gives raw rank 96 over C for the
pulled-back model, and still does not decide Candidate A versus Candidate B. Topology can
also fail if the symbol is non-elliptic or background-dependent through `ch_2(F)`.

Synthesis / aufhebung: preserve the K3/APS route, but move the live invariant from raw fiber
rank to the K-theory symbol class of the physical RS complex.

Closure / falsification condition: close if the symbol-index contract outputs Candidate A,
Candidate B, or another computed index under non-circular rules. Falsify the current
generation-count story if the operator is non-elliptic, depends on unchosen background Chern
data, or gives an incompatible index.

### Priority verdict

lead

## Persona 05: General Relativist

### Missing or underweighted now

The repo is underweighting exact nonlinear GR recovery. The weak-field Schwarzschild result
is useful, but it is not exact Schwarzschild or Kerr. The action notes say the right thing:
without a written variational principle and an IG variation rule, exact black-hole recovery
and theta dark-energy dynamics remain open.

The IG dynamics note adds a serious obstruction: a bare `|theta|^2` term with freely varied
`beta` forces `theta = 0`. That blocks the dark-energy source story unless IG variables are
background/Stueckelberg data, constrained variables, or dynamical fields with their own terms.

### Steelman

Write and test the minimal GU action branch. Derive the full Euler-Lagrange tuple for
`(s, A, eps, beta, Psi)`, resolve the IG variation status, then run two binary reductions:
exact Schwarzschild/Kerr as vacuum solutions, and FLRW theta reduction extracting the
canonical `xi R theta^2` coefficient.

### Hegelian path against the steelman

Thesis: a genuine unification must reproduce exact GR solutions and cosmological dynamics
from a single action.

Antithesis: the current action is under-specified; Willmore-only exact Schwarzschild is an
obstruction; free IG variation kills nonzero theta; the DESI-sign theta mechanism needs a
negative nonminimal curvature coupling that has not been derived from GU geometry.

Synthesis / aufhebung: preserve the weak-field and theta reconstructions, but demote them to
tests of a written action rather than evidence that the action already works.

Closure / falsification condition: pass if the written action admits exact Schwarzschild and
Kerr vacuum configurations and derives `xi_eff < -0.319` in the stated normalization. Fail if
either exact vacuum GR remains obstructed or the reduced theta mode lacks the needed coupling.

### Priority verdict

lead

## Persona 06: QFT Theorist

### Missing or underweighted now

The repo is underweighting the difference between a classical geometric construction and a
4D quantum field theory. VZ is a principal-symbol and dynamical-consistency gate, not just
an algebraic curiosity. Type II_1 hosting does not yet address extra modes, anomaly shadows,
or the full fluctuation orbit. Observer/finality CHSH fixtures do not become physics until a
GU-derived state and admissible observables exist.

### Steelman

After the action is written, derive the 4D EFT data from it: propagating degrees of freedom,
constraints, anomaly content, VZ hyperbolicity/constraint propagation, loop-sensitive B/C
non-decoupling, and the domain in which the higher-dimensional theory is an EFT rather than
a claimed UV-complete QFT.

### Hegelian path against the steelman

Thesis: QFT consistency is the final arbiter of chirality, spin-statistics, anomalies,
hyperbolicity, and physical measurement predictions.

Antithesis: the repo is not yet ready for full QFT evaluation because the action and field
content are under-specified, while several current results are only principal-symbol,
fixture, or reconstruction grade.

Synthesis / aufhebung: let QFT act as a protective layer: it should not block the classical
action work, but every positive classical claim must name the QFT gate it would later face.

Closure / falsification condition: pass if the written action yields a consistent low-energy
4D EFT with the stated anomaly and hyperbolicity controls. Fail if VZ dynamics, anomaly
shadows, or extra modes survive in a way incompatible with SM+GR.

### Priority verdict

protect

non-smuggling positive gate rather than broad interpretive synthesis.

Closure / falsification condition: pass if the repo derives the generation count and SM
representation content without target-number inputs. Fail if the only available derivation
uses physical DOF counting, externally supplied CC modules, or a chosen order-3 object.

## Group-Level Convergence Map

### Where the 10 agree

All ten lenses accept that `Y^14` as a 4+10 metric-bundle object is mathematically meaningful.
None treats that fact as a derivation of SM+GR.

They converge on proof-grade discipline:

- the RS generation count is open until the constrained/gauge-fixed K3 symbol class and
  index are computed;
- VZ is promising but conditional until the typed rolled-up operator and Q-sector E-block
  are proof-grade;
- Type II_1 is an open hosting/construction lane, not an explanatory selector yet;
- observer/finality has formal value, but physical CHSH forcing is open until GU supplies
  `rho_AB` and admissible observables;
- weak-field Schwarzschild is not exact nonlinear GR recovery;
- dark-energy theta dynamics depends on the written action and the IG variation status.

### Where they disagree

The differential geometer, spinor specialist, and mathematical physicist want typed
definitions first. The algebraic topologist wants the RS symbol-index computation first.
The general relativist wants the action and exact 4D physics gate first. The gauge theorist
and representation theorist want selector discipline first. The QFT theorist wants all of
these treated as pre-QFT classical gates. The KK specialist wants the repo to stop borrowing
compactification intuitions from the wrong comparison class.

This is not a real contradiction. It is a dependency graph:

1. Typed geometry and action conventions are shared prerequisites.
2. RS symbol-index and VZ proof-grade checks are the most important mathematical gates.
3. Gauge/representation/Type II_1/observer claims are downstream unless they produce a
   non-smuggling selector or GU-derived physical state.

### Group recommendation

Lead with two proof-grade spines in parallel: the written GU action with IG dynamics and
exact 4D reduction gates, and the RS constrained/gauge-fixed K3 symbol-index computation.
Protect VZ, Type II_1, and observer/finality as conditional lanes with strict no-overclaim
rules. Park cardinality-only generation selectors and CHSH physical-forcing claims until
they defeat their anti-smuggling or GU-derived-state gates. Reject any generation-count or
SM-emergence claim that uses the target number, embedded CC data, or a hand-chosen threefold
object as an input.
