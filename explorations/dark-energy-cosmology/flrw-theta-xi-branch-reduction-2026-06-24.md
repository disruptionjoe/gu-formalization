# FLRW Theta-Xi Branch Reduction

## 1. Verdict

No currently written viable action branch generates negative

```text
xi_eff = C_Rtheta / Z_theta.
```

The branch verdict is:

```text
negative xi generated:  absent from current evidence
strong viable branches: branch-undefined for xi_eff, because the action data needed
                        to compute Z_theta and C_Rtheta is not yet fixed
bare free-beta branch:  theta mode absent; xi_eff undefined because theta = 0
bare inserted xi:       inconsistent with GU-provenance test unless primary-sourced
```

This is stronger than "not yet numerically evaluated" but weaker than "proved zero." The
minimal action deliberately sets the bare 4D curvature term to zero:

```text
S_Rtheta,bare = 0.
```

Therefore a negative coefficient can count only if it is generated by the reduction of the
written GU action. Branches 1, 2A, 2B, and 3 do not yet supply enough action/constraint data
to calculate that generated coefficient. Branch 4 supplies enough data, but it kills the
theta mode.

The dark-energy mechanism note identifies the necessary target:

```text
xi_eff < -0.319       for the negative-w_a window
xi_eff ~= -0.6        for the reconstruction-grade DESI ratio match
```

No branch below reaches that target from GU geometry.

## 2. Viable Action Branches Considered

I use the branch names and sign conventions of the closed-loop action gate:

```text
theta = A - Gamma(eps) - Ad(eps^-1) beta
S_theta = -c_theta/2 int_Y <theta, theta>
delta_beta theta = -Ad(eps^-1) delta beta
```

The considered branches are:

| Branch | Variation status | Source law status | FLRW theta-xi status |
|---|---|---|---|
| 1. Background/Stueckelberg | `eps,beta` not varied | preserves `D_A^*F_A = theta` after normalization | branch-undefined |
| 2A. A-independent constrained IG | `Phi(eps,beta,s)=0`, `D_A Phi=0` | preserves `D_A^*F_A = theta` | branch-undefined |
| 2B. A-dependent constrained IG | `Phi(A,eps,beta,s)=0` | corrected by multiplier current | branch-undefined |
| 3. Dynamical IG / total current | kinetic or first-order `S_IG-dyn` | replaces source with `theta_eff` | branch-undefined |
| 4. Bare free-beta norm | free `beta`, only `S_theta` | collapses to `D_A^*F_A=0` | theta mode absent |

Branch 2A remains the strongest conservative branch for preserving the bare source equation,
but it is not yet a coefficient-producing branch because `Phi` has not been derived. Branch 3
is the strongest physical dark-energy branch, but it is not yet a coefficient-producing branch
because `S_IG-dyn`, its current `J_IG`, and its scalar projection are not fixed.

## 3. FLRW Theta Scalar Reduction

The FLRW ansatz required by the theta dark-energy files is:

```text
g = -dt^2 + a(t)^2 gamma_ij dx^i dx^j
Psi = 0
s^*theta = b_raw(t) u_0
```

where `u_0` is a homogeneous isotropic internal/pullback mode normalized by the same pairing
used in the action. The reduction must produce a four-dimensional quadratic action of the
form

```text
S_eff[b_raw,g] =
  int_X sqrt(-g) [
    -1/2 Z_theta g^mu nu partial_mu b_raw partial_nu b_raw
    -1/2 (C_M + C_Rtheta R[g]) b_raw^2
    + ...
  ].
```

If `Z_theta > 0`, the canonical scalar is

```text
B = sqrt(Z_theta) b_raw
```

and the canonical action reads

```text
S_eff[B,g] =
  int_X sqrt(-g) [
    -1/2 g^mu nu partial_mu B partial_nu B
    -1/2 (M_KK^2 + xi_eff R[g]) B^2
    + ...
  ],

xi_eff = C_Rtheta / Z_theta.
```

If `Z_theta = 0`, there is no propagating scalar normalization. If the branch forces
`b_raw = 0`, then the theta dark-energy scalar is absent and `xi_eff` is not a meaningful
number.

### Branch 1: Background/Stueckelberg IG

Here

```text
theta = A - C_background,
C_background = Gamma(eps) + Ad(eps^-1) beta,
delta eps = delta beta = 0.
```

The only possible scalar is a connection-distortion component:

```text
s^*theta = b_raw(t) u_0.
```

The bare norm `S_theta` supplies an algebraic mass-like term for `b_raw`; it does not by
itself supply a scalar kinetic term or a curvature coupling. Kinetic and curvature terms, if
present, must come from reducing `S_YM[A]`, `S_W[s,A,eps,beta]`, and any section-pullback
couplings in the same normalization.

Therefore the branch-level reduction stops at:

```text
scalar mode:  conditional ansatz, connection-distortion mode
Z_theta:      undefined until the YM/section projection onto u_0 is computed
C_Rtheta:     undefined; no bare R[g] |s^*theta|^2 term is present
xi_eff:       undefined
verdict:      negative xi not generated by current branch data
```

A truncation to `S_theta` alone is even weaker:

```text
Z_theta = 0,
C_Rtheta = 0,
xi_eff undefined because there is no canonical scalar.
```

That truncation cannot support the FLRW theta dark-energy mechanism.

### Branch 2A: A-Independent Constrained IG

The action is schematically

```text
S = S_YM[A] + S_theta[A,eps,beta] + S_W[s,A,eps,beta]
    + int_Y <lambda, Phi(eps,beta,s)>.
```

with

```text
D_A Phi = 0.
```

The beta equation becomes

```text
c_theta Ad(eps) theta + (D_beta Phi)^* lambda = 0,
```

so it projects theta against the tangent space of the constraint rather than forcing
`theta=0`, provided the allowed beta variations are genuinely smaller than the full
`Omega^1(Y,ad P)` space.

On FLRW, the scalar mode is not simply "any component of theta." It must survive the
constraint linearization:

```text
D_beta Phi(delta beta) + D_eps Phi(delta eps) + D_s Phi(delta s) = 0.
```

Only the unconstrained residual direction can be used as

```text
s^*theta = b_raw(t) u_0.
```

Since `Phi`, its tangent space, and the multiplier solution are not yet derived, neither the
allowed scalar direction nor its quadratic action is fixed.

Thus:

```text
scalar mode:  branch-undefined until Phi and T C_IG are explicit
Z_theta:      branch-undefined
C_Rtheta:     branch-undefined
xi_eff:       branch-undefined
verdict:      no negative xi generated; coefficient cannot be read off
```

Branch 2A preserves the source equation, but preservation of the source equation is not a
curvature-coupling computation.

### Branch 2B: A-Dependent Constrained IG

If

```text
Phi = Phi(A,eps,beta,s),
```

then the connection equation becomes

```text
g_A^-2 D_A^*F_A - c_theta theta + (D_A Phi)^* lambda = 0.
```

The FLRW scalar action now receives possible multiplier contributions from

```text
delta^2 int_Y <lambda, Phi(A,eps,beta,s)>.
```

Those terms can modify both the kinetic normalization and the curvature coefficient. For
example, a constraint of the schematic form `D_A^*theta=0` can enforce a divergence condition,
but it also inserts derivative multiplier terms into `E_A`.

Because neither `Phi(A,eps,beta,s)` nor the multiplier branch is written, the reduction is:

```text
scalar mode:  branch-undefined
Z_theta:      branch-undefined; multiplier terms may contribute
C_Rtheta:     branch-undefined; multiplier terms may contribute
xi_eff:       branch-undefined
verdict:      no negative xi generated; source law is also no longer bare theta
```

Branch 2B is not inconsistent, but it cannot be used to claim the old source equation or a
negative `xi_eff` without a full multiplier reduction.

### Branch 3: Dynamical IG / Total Current

A representative second-order branch is

```text
U = Ad(eps^-1) beta,
theta = A - Gamma(eps) - U,

S_IG-dyn =
  -Z_U/2 int_Y |D_A U|^2
  -int_Y V(U,eps)
  -c_theta/2 int_Y |theta|^2.
```

A first-order parent version replaces the kinetic term by

```text
int_Y <P, D_A U> - 1/(2Z_U) int_Y |P|^2.
```

This branch can produce a genuine scalar mode because the IG equation is differential rather
than algebraic:

```text
E_U:
  c_theta theta - Z_U D_A^*D_A U - V_U = 0
```

or, in first-order form,

```text
E_U:
  -D_A^*P + V_U + c_theta theta = 0.
```

But it also changes the source equation:

```text
D_A^*F_A = theta_eff,
theta_eff = c_theta theta - J_IG.
```

On FLRW, write the surviving scalar component as

```text
U = u_background + b_raw(t) u_0 + ...
theta = theta_background + b_raw(t) theta_0 + ...
```

Then the coefficient calculation has the schematic structure

```text
Z_theta =
  Z_U * ||D_b U||^2_fiber
  + Z_A/YM contribution from the A-theta relation
  + possible section/current terms,

C_Rtheta =
  curvature term from the covariant Laplacian/Weitzenbock reduction
  + curvature term from the section-pullback geometry
  + curvature term from written cross-couplings, if primary-sourced.
```

This is the right branch shape for a dark-energy scalar, but not yet a computation. The
numbers depend on the actual `S_IG-dyn`, representation, projection measure, potential,
background solution, and current term.

Therefore:

```text
scalar mode:  plausible, but branch-undefined until S_IG-dyn and u_0 are fixed
Z_theta:      branch-undefined
C_Rtheta:     branch-undefined
xi_eff:       branch-undefined
verdict:      no negative xi generated; strongest route to compute it later
```

If this branch is chosen, the canon source language must be rewritten around `theta_eff`.
Conservation belongs to the total current, not generally to bare theta.

### Branch 4: Bare Free-Beta Norm

This branch is fully reducible enough to fail:

```text
S = S_YM[A] - c_theta/2 int_Y |theta|^2 + S_W,
delta beta arbitrary.
```

The beta equation is

```text
E_beta = c_theta Ad(eps) theta = 0,
```

hence

```text
theta = 0,
s^*theta = 0,
b_raw = 0.
```

Thus:

```text
scalar mode:  absent
Z_theta:      no physical theta scalar normalization
C_Rtheta:     no physical theta curvature coefficient
xi_eff:       undefined, not zero as a dark-energy scalar coefficient
verdict:      negative xi absent because the theta mode is killed
```

Adding a bare `R[g] |s^*theta|^2` term after this does not save the branch. The branch was
defined by freely varying beta with only the bare norm as the IG term; changing the variation
status or adding dynamics moves it to another branch.

The table should be read as a gate, not as a parameter fit. A future worker may fill in
`Z_theta` and `C_Rtheta` only from a written branch action and scalar projection. They may not
promote the phenomenological value `xi ~= -0.6` into GU provenance.

## 5. What This Means For The Dark-Energy Mechanism

The FLRW mechanism search found a clean cosmology fact:

```text
negative w_a is reachable if xi_eff < -0.319,
with xi_eff ~= -0.6 matching the DESI ratio at reconstruction grade.
```

This branch reduction says the GU action provenance is still missing. Minimal `xi=0` does not
give the DESI sign, conformal `xi=+1/6` does not give the DESI sign, and none of the current
GU action branches derives the needed negative coefficient.

Consequences:

```text
THETA-XI status: remains open for GU provenance
dark-energy scalar canon: remains conditional on scalar character of s^*theta
negative-w_a route: phenomenologically alive, not GU-derived
Branch 4 actual action: theta dark energy fails
Branch 2A future path: derive Phi, then reduce
Branch 3 future path: write S_IG-dyn, then reduce and revise source law
```

The project should not say "GU generates negative xi" yet. The accurate statement is:

```text
The only identified FLRW mechanism that reaches the DESI-sign window is a sizable negative
non-minimal curvature coupling, and the current GU action/IG branches have not generated it.
```

## 6. Next Meaningful Proof/Computation Step

The next step should not be another cosmology scan. The decisive computation is geometric:

1. Choose the surviving action branch.

   Prefer Branch 2A only if a primary or geometric source derives

   ```text
   Phi(eps,beta,s)=0,
   D_A Phi = 0,
   T C_IG != full beta variations.
   ```

   Otherwise use Branch 3 and write the actual `S_IG-dyn`.

2. Define the FLRW scalar projection:

   ```text
   s^*theta = b_raw(t) u_0,
   <u_0,u_0> and fiber/normal measure fixed.
   ```

   Also prove that `u_0` is scalar, not spin-2 or constrained away.

3. Expand the written action to quadratic order in `b_raw` on FLRW:

   ```text
   S_GU = S_background + S_2[b_raw,g] + O(b_raw^3).
   ```

4. Read off the actual coefficients:

   ```text
   Z_theta      = coefficient of g^mu nu partial_mu b_raw partial_nu b_raw
   C_Rtheta     = coefficient of R[g] b_raw^2
   xi_eff       = C_Rtheta / Z_theta
   ```

5. Compare only after canonical normalization:

   ```text
   xi_eff < -0.319       pass for negative-w_a window
   xi_eff ~= -0.6        reconstruction-grade DESI-ratio match
   xi_eff >= -0.319      DESI-sign mechanism absent in that branch
   no scalar mode         theta-FLRW KG mechanism inapplicable
   ```
