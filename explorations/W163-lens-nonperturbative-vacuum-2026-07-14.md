## 1. Persona 1 -- non-perturbative QFT theorist: the Coleman-Weinberg false-vacuum signature

The cleanest test of the lens premise is Coleman-Weinberg. W126 proved the tree potential on the
scale mode is EXACTLY quadratic, `W(u) = -64 u^2 - 8 u + 2` (degree 2, `c_3 = c_4 = ... = 0`,
reproduced NPV3a): there is no tree minimum, so the ONLY candidate for a non-perturbative minimum is
the one-loop correction. The standard one-loop effective potential for a real scalar of
field-dependent mass `M^2` is `V_1 = (1/64 pi^2) M^4 [ln(M^2/mu^2) - 3/2]` (Coleman-Weinberg,
PORTED). Evaluate it at the GU native scalaron mass `M^2 = m_0^2 = -1/4 < 0` (W130). Because the
argument of the logarithm is NEGATIVE, `ln(M^2/mu^2) = ln(|M^2|/mu^2) + i pi`, and the effective
potential acquires an imaginary part:

```
Im V_eff = (1/64 pi^2) M^4 * pi = M^4/(64 pi) = (1/16)/(64 pi) = 1/(1024 pi) != 0    (NPV3b)
```

A nonzero `Im V_eff` is exactly the Callan-Coleman false-vacuum decay width: it is PRESENT precisely
because the scalaron is tachyonic (`M^2 < 0`), and it VANISHES when `M^2 >= 0` (NPV3c). This is the
decisive positive result of the lens: **the perturbative point IS a false vacuum, computed, at GU's

the imaginary part does NOT do is locate that true vacuum -- it only certifies the departure. That is
the honest boundary: the false-vacuum signature is a GO for the search, not the destination.
