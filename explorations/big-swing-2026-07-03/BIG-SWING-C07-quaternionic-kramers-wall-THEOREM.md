# C-07: does the quaternionic Kramers wall forbid an odd literal generation count?

**The swing.** GU's generation carrier is the 1664-dimensional gamma-traceless slice `W`
(`1792 - 128 = 1664`, EVEN) of `V (x) S`, with `Cl(9,5) = M(64,H)` acting. Because
`p - q = 9 - 5 = 4 mod 8`, the real Clifford algebra sits in the **quaternionic** class `M(64,H)`, and
the spinor module carries an **antiunitary** structure `J_quat = id_14 (x) U` with `U` antilinear and
`U^2 = -I` (verified exact to ~`2.2e-14`). The prove-or-break target: *does this `J^2 = -1` structure
force EVERY GU-native self-adjoint operator on `W` to have **EVEN** metric signature
`sig(A) = n_+(A) - n_-(A)`, so the literal generation index (`= signature`) can **never be odd**, in
particular **never 3**?*

**Honest outcome: a THEOREM â€” a conditional no-go.** Over the full admissible GU-native class, and under
the two explicitly declared hypotheses, the answer is **yes, the signature is always even**. No trap-free,
target-free, GU-native operator produces an odd signature; every candidate that reaches an odd number does
so only by importing non-quaternionic structure by hand (the named Trap 4) or lands right back on an even
count. The wall stands as:

> **No GU-native, H-class completion can force an odd literal generation count.**

This is **publishable** as a conditional no-go. It is **not** "three generations is impossible" â€” see
*What this settles, and what it does not*.

## The wall, precisely (class + hypotheses stated up front)

**Hypotheses (program inputs, not theorems).**
- **(H1) Signature.** The metric is the H-class `(9,5)`, `p - q = 4 mod 8`, so `J^2 = -1` (verified exact).
  Under `(7,7)` one would get `p - q = 0 mod 8`, `J^2 = +1`, the **real** class `M(128,R)`, Kramers
  dissolves, and odd-dimensional real irreps are unobstructed â€” the wall **disappears**.
- **(H2) Literal-index reading.** `count = signature (= index)`. Under the Kramers-doubled `count = index/2`
  reading a rank-`r` carrier gives `count = r` with `r` free, so `3` becomes reachable by
  under-determination.

**Admissible class (two independently sufficient formalizations; the wall holds over EACH).**
1. **H-LINEAR generated algebra** â€” every self-adjoint operator in the real algebra generated by GU's
   a-priori primitives (`e_a` incl. timelike `i*G_a`, bivectors `sigma_ab`, chirality `omega = Gamma_15`,
   vector-index `M_ij` in the `M(14,C)` factor, `Pi_RS`/`Q`, twisted Dirac `M_D = id_14 (x) c(xi)`, plus the
   full BV/BRST/ghost apparatus). Every primitive is Clifford-built or lives in `M(14,C)`, so the whole
   generated algebra sits inside `M(14,C) (x) M(64,H)`, which by the double-commutant theorem **is**
   `commutant(J_quat)` â€” H-linearity holds by construction (verified numerically to ~`1e-11`).
2. **Spin(9,5)-EQUIVARIANT C-linear algebra** â€” every self-adjoint operator commuting with the Spin(9,5)
   action, acting as `id_{V_i} (x) a_i` on each isotypic component.

**Critical distinction (do not conflate).** H-linearity and Spin-equivariance are **different** conditions,
neither implying the other: `e_a`, `M_D` are H-linear but not equivariant (`[e_0, sigma] ~ 11.3`); scalar
`i*id` is equivariant but not H-linear (J-defect ~`22.6`). The wall holds over **both** by two different
arguments, so the operative hypothesis is whichever a candidate claims. An operator that is **neither**
(e.g. an arbitrary rank-3 kernel projector) is a **foreign import**, outside the class.

**The genuine gap (the load-bearing unproven premise).** Nothing in the reconstruction forces the
**physical** generation carrier to lie in this GU-native class rather than being an arbitrary self-adjoint
operator on the 1664 slice. The theorem constrains the class; it does not prove the physical carrier is in
the class.

## The proof (rigorous, over the full class)

Two independent closed-form arguments, one per formalization, plus a table-free hardening. All three were
reproduced numerically here (`scratchpad/c07_verify.py`, `scratchpad/c07b.py`).

**(A) Kramers â€” H-linear branch.** *Lemma.* If `J` is antilinear with `J^2 = -I` and `A = A^dagger`
commutes with `J`, then for every real eigenvalue `lambda`, `J` maps `E_lambda` to itself antilinearly with
`J^2 = -I`, so `E_lambda` is a quaternionic module and `dim_C E_lambda` is **even**; hence `n_+(A)` and
`n_-(A)` are each even and `sig(A) = n_+ - n_-` is **even**. *Containment.* The entire GU-native generated
algebra lies in `M(14,C) (x) M(64,H) = commutant(J_quat)` (`Cl(9,5) = M(64,H)` is exactly the
quaternion-linear algebra), so **every** H-linear GU-native `A` commutes with `J` and inherits even
signature. This is the surviving proof, graded **RIGOROUS**, `full_class = true`, `survived_hunt = true`.

**(B) Quaternionic-isotypic â€” equivariant branch (needs NO commuting-with-J).** An equivariant Hermitian
`A` acts as `id_{V_i} (x) a_i`, so `sig(A) = sum_i sig(a_i) * dim_C(V_i)`. Every Spin(9,5)-irrep inside an
`M(64,H)`-module is quaternionic / H-paired, so `dim_C(V_i)` is **even**, forcing every term â€” hence
`sig(A)` â€” even. The 1664 slice is a Spin-summand of an `M(64,H)`-module, so both (A) and (B) apply.

**(C) Table-free hardening (parity identity).** On the EVEN 1664 slice, for **any** Hermitian `A`,
`n_+ + n_- = 1664 - nullity`, so `sig = n_+ - n_- == nullity (mod 2)` (verified). Odd signature therefore
**requires odd nullity**. But GU-native (H-linear / equivariant) kernels are even-dimensional
(quaternionic), so nullity is even and signature is even **regardless of J**. This closes the wall without
any irrep census.

**Numerical corroboration (reproduced this session).**

```
--- Random H-linear (J-commuting) Hermitian on 2n=16: expect EVEN signature ---
  J-defect=0.00e+00  signature=0  (even=True)  nullity=0     [x6, all sig 0]

  rank-3 projector (foreign): sig=3 nullity=13 -> ODD sig from ODD nullity

```

The only way to an odd signature on the even slice is odd nullity, and the only odd-nullity object anywhere
in the tournament is a **hand-imported rank-3 projector** â€” which is not GU-native.

**Honesty note on the mod-3 phrasing.** The source's "all forced counts are `0 mod 3`" is imprecise
(`832 = 2^6 * 13` is `1 mod 3`; `1664 = 2^7 * 13` is `2 mod 3`). The load-bearing invariant is **2-primary
EVEN parity**, which is what makes any odd value (including 3) unreachable. Do not lean on the mod-3
phrasing.

## Why every break failed (no survivor)

Four break families were attempted; **all** self-graded `FAILS_TO_BREAK` and none survived scrutiny. They

  at J-defect `~1e-11`. **Does not flip parity.**

Crucially, no break tripped the count via a legitimate route. The two moves "on the table" in canon each

## What this settles, and what it does not

**Settles (this swing).** Over the full admissible GU-native class, in **both** formalizations, under H1 and
H2, `sig(A)` is provably EVEN â€” the Kramers double-commutant argument (A) and the quaternionic-isotypic
argument (B) both hold, corroborated table-free by (C) and to machine precision numerically, and **no
trap-free target-free break exists**. The conditional statement â€” *"odd literal signature is impossible over
the H-class GU-native self-adjoint class"* â€” is **SOUND** and **publishable** as a no-go:
**no GU-native H-class completion can force an odd literal generation count.**

**Does NOT settle (and no target imported).** This does not derive three and does not forbid three. Three
residual gaps keep the generation-count verdict **OPEN**:
1. **Physical-carrier-in-class (the `S_IG` selector gap).** Nothing forces the physical carrier into the
   GU-native class; an arbitrary self-adjoint operator on the 1664 slice is outside the theorem's reach.
2. **Signature is a declared input.** `(9,5)` vs `(7,7)` (`J^2 = -1` vs `+1`) is a program input the algebra
   does not settle; a `(7,7)` derived as GU-native would dissolve the wall.
3. **Count reading.** Under `count = index/2` the rank/count is free, so 3 is reachable by
   under-determination.

The forbidden targets (`3`, `chi(K3) = 24`, `8`, `16`, `Ahat = 3`, `rank_H = 4`, any hand-chosen rank) were
never inserted, divided into, or constructed toward. A break that only reopens the count by relaxing H1 or
H2 proves **under-determination**, not a forcing of 3.

**Where the wall now is (what would reopen the count).** A genuine break must (i) settle `(7,7)` as
GU-native output, or (ii) build the definite internal-fiber (K3 / `GL(4,R)/O(3,1)`) family Dirac source from
GU-forced data â€” target-free, no smuggled constant â€” and show it forces a nonzero count *without* abandoning
chirality, or (iii) prove the physical carrier must lie in the GU-native class (closing gap 1 the other way,
which would make the no-go unconditional). Absent one of these, the parity wall is closed and the
generation-count verdict remains **external, located but not forced**.
