# Closed-Loop GU Action / IG Branch Gate

## Verdict

The action/IG lane now has a branch decision, but not a physics upgrade.

No branch currently derives

```text
xi_eff < -0.319
```

without inserting a curvature coupling. The only known FLRW mechanism that reaches
the DESI-sign window is a sizable negative non-minimal term

```text
-1/2 xi R[g] B^2,     xi roughly -0.6,
```

but none of the current IG/action branches derives that coefficient from the written
GU geometry. The exact Schwarzschild/Kerr gate also remains unpassed: the weak-field
Schwarzschild canon is a bounded `O(M/r)` pass, while exact nonlinear Schwarzschild
and Kerr require the full coupled section equation to cancel the known Willmore
residual. No branch below supplies that cancellation yet.

Recommended strongest branch:

```text
Branch 2A: A-independent constrained IG / Stueckelberg geometry.
```

It is the best conservative branch because it preserves the advertised vacuum source
equation

```text
D_A^* F_A = theta
```

while avoiding the free-beta collapse `theta = 0`. Its upgrade is blocked by one
proof obligation:

```text
Derive the allowed variation space / constraint Phi(eps,beta,s)=0 from a written
GU action or from the tau^+ / IG geometry, then compute the full EL tuple and
FLRW xi_eff in that same normalization.
```

If no primary or geometric constraint exists, the honest fallback is Branch 3:
dynamical IG with a total-current law, not the bare theta-source equation.

## Notation and Shared Baseline

Use the minimal-action notation from the action spec:

```text
A        Sp(64) connection on P -> Y
F_A      curvature of A
eps      IG / gauge variable
beta     IG translation field
U        Ad(eps^-1) beta
Gamma    reference connection Gamma(eps)
theta    A - Gamma(eps) - U
s        section X^4 -> Y^14, with g = s^*gY
B_s      II_s^H, horizontal-normalized section second fundamental form
```

The baseline branch action is schematically:

```text
S_0 = S_YM[A] + S_W[s,A,eps,beta] - c_theta/2 int_Y <theta,theta>.
```

With the sign convention of `gu-minimal-action-spec`, the connection variation gives

```text
E_A =
  g_A^-2 D_A^* F_A
  - c_theta theta
  + E_A^W
  + J_Psi
  + E_A^IG-extra
  + E_A^cross
  = 0.
```

In vacuum, ignoring terms that are absent in a chosen branch, and normalizing
`g_A^2 c_theta = 1`, this becomes the transcript-style source equation:

```text
D_A^* F_A = theta.
```

The crucial beta variation is branch-deciding:

```text
delta_beta theta = - Ad(eps^-1) delta beta

delta_beta S_theta
  = c_theta int_Y <Ad(eps) theta, delta beta>.
```

If `delta beta` is free, then

```text
E_beta = c_theta Ad(eps) theta = 0
```

and nonzero theta is killed. Every viable branch must therefore explain why this
free auxiliary-field equation is absent, constrained, or replaced by real IG dynamics.

## Exact Physics Gates Used Here

### Gate A: Exact Schwarzschild/Kerr

A branch passes only if it produces a smooth vacuum configuration

```text
(s_g, A_g, eps_g, beta_g, Psi_g = 0)
```

for both Schwarzschild and Kerr such that the full EL tuple vanishes:

```text
E_s = E_A = E_eps = E_beta = E_Psi = 0.
```

Passing weak-field Schwarzschild at `O(M/r)` is not enough. The branch must cancel or
remove the exact section residual from the Willmore equation without relabeling
embedding stress as matter in a vacuum test.

### Gate B: FLRW `xi R theta^2`

A branch passes only if the same action reduces to a canonical homogeneous scalar mode

```text
S_eff contains int_X sqrt(-g) [
  -1/2 (partial B)^2
  -1/2 (M_KK^2 + xi_eff R[g]) B^2
  - Lambda_eff
  + ...
]
```

with

```text
xi_eff = C_Rtheta / Z_theta < -0.319
```

and the coefficient is generated by the GU action, not inserted as a tunable bare
`R theta^2` term.

## Branch 1: Background / Stueckelberg IG Variables

### Definition

Declare the IG variables fixed in the action principle:

```text
delta eps = 0,
delta beta = 0.
```

They may still transform as Stueckelberg spurions so that `theta` transforms
covariantly, but they do not produce EL equations.

### EL Consequences

There is no `E_beta` equation, so the algebraic collapse is avoided:

```text
theta != 0   allowed.
```

The vacuum connection equation remains:

```text
g_A^-2 D_A^* F_A - c_theta theta = 0,
```

or, after normalization,

```text
D_A^* F_A = theta.
```

The divergence-free status is weaker than the old canon slogan. If `(eps,beta)` are
fixed spurions, the Noether identity can include omitted background variations. Thus

```text
D_A^* theta = 0
```

is not automatic from background status alone. It must come either from a separately
gauge-invariant theta sector with a valid sector-wise Noether identity, or from a
verified source-equation integrability identity. The local notes explicitly warn that
`D_A^* D_A^* F_A = 0` is not a generic algebraic identity one can assume blindly.

### Exact Schwarzschild/Kerr

Branch 1 does not pass the exact black-hole gate as written.

Reason: it preserves the source equation, but it does not derive the section equation
contributions

```text
delta_s S_YM + delta_s S_theta + delta_s S_W
```

well enough to cancel the known exact Schwarzschild/Kerr Willmore residual. It keeps
the weak-field `O(M/r)` Schwarzschild result intact, but exact nonlinear vacuum
recovery remains open.

### FLRW `xi R theta^2`

Branch 1 does not derive `xi_eff < -0.319`.

Any FLRW scalar mode would have to come from the connection/section reduction of
`theta = A - C_background`. The branch has not produced the canonical kinetic term,
the wavefunction factor `Z_theta`, or the curvature coefficient `C_Rtheta`. If
`xi_eff` is set by hand, the branch fails the no-insertion rule.

### Outcome

```text
nonzero theta:        PASS
source equation:      PASS for D_A^*F_A = theta
exact Schwarzschild:  OPEN/FAIL at current evidence
exact Kerr:           OPEN/FAIL at current evidence
FLRW xi gate:         FAIL/UNPROVED; no xi_eff derived
```

Branch 1 is source-equation clean but physically thin: theta survives as distortion
relative to fixed data, not as a fully dynamical IG field.

## Branch 2: Constrained IG Variables

### Definition

Vary `(eps,beta)` only inside a constrained IG submanifold:

```text
Phi_I(eps,beta,s) = 0.
```

Add multipliers:

```text
S = S_0 + int_Y <lambda^I, Phi_I(eps,beta,s)>.
```

The branch splits into two subcases.

```text
Branch 2A: Phi is independent of A.
Branch 2B: Phi depends on A.
```

### EL Consequences: A-Independent Constraint

For Branch 2A:

```text
E_A:
  g_A^-2 D_A^* F_A - c_theta theta = 0.

E_beta:
  c_theta Ad(eps) theta + (D_beta Phi)^* lambda = 0.

E_eps:
  c_theta L_eps^* theta + (D_eps Phi)^* lambda = 0.

E_lambda:
  Phi(eps,beta,s) = 0.
```

The beta equation no longer implies `theta = 0` if the allowed beta variations do
not fill the whole beta-field space. It only says that `theta` is orthogonal, in
the variational pairing, to the tangent directions of the constraint surface.

Therefore nonzero theta is allowed when:

```text
theta lies in the normal/cokernel directions of D_beta Phi.
```

The connection equation is unchanged from the baseline:

```text
D_A^* F_A = theta.
```

This is the strongest way to preserve the current source equation without hiding the
free-beta collapse.

### EL Consequences: A-Dependent Constraint

If the constraint depends on `A`, then:

```text
E_A:
  g_A^-2 D_A^* F_A - c_theta theta + (D_A Phi)^* lambda = 0.
```

So the source equation becomes:

```text
D_A^* F_A = theta - g_A^2 (D_A Phi)^* lambda.
```

For example, imposing `Phi = D_A^* theta = 0` protects divergence-free theta by
construction, but it adds derivative multiplier terms to `E_A`. This is no longer
the bare equation `D_A^*F_A = theta` unless the multiplier term vanishes on the
specific solution branch.

### Exact Schwarzschild/Kerr

Branch 2A does not yet pass exact Schwarzschild/Kerr.

It is the most conservative branch for the connection equation, but the exact
black-hole gate is a section-equation gate. The branch must still compute:

```text
E_s =
  delta_s S_W
  + delta_s S_YM
  + delta_s S_theta
  + delta_s S_constraint
  = 0
```

on Schwarzschild and Kerr. The constraint must supply an internal vacuum cancellation
of the Willmore residual, not merely allow a nonzero theta. No such cancellation has
been derived.

Branch 2B is even less clean for exact black holes because the multiplier current in
`E_A` must also be solved and interpreted without smuggling matter stress.

### FLRW `xi R theta^2`

Branch 2 has not derived the negative curvature coupling.

The only plausible route is that the constrained geometry

```text
beta = beta_0(eps,s)
```

or a tau-plus/double-coset slice makes the pullback `s^*theta` acquire a canonical
scalar action with:

```text
xi_eff = C_Rtheta / Z_theta < -0.319.
```

But until `Phi` is explicit, the coefficient is undefined. An A-independent
constraint preserves the source equation; it does not by itself produce an
`R[g] B^2` term.

### Outcome

```text
nonzero theta:        PASS if constraint removes free beta variations
source equation:      PASS for 2A, CORRECTED_SOURCE for 2B
exact Schwarzschild:  OPEN/FAIL at current evidence
exact Kerr:           OPEN/FAIL at current evidence
FLRW xi gate:         FAIL/UNPROVED; xi_eff undefined until reduction
```

Branch 2A is the recommended branch because it is the only branch that is both
nonzero-theta viable and source-equation conservative.

## Branch 3: Dynamical IG / Total Current

### Definition

Promote the IG variables to real dynamical fields. A representative second-order
version is:

```text
S = S_YM[A]
    - c_theta/2 int_Y |theta|^2
    - Z_U/2 int_Y |D_A U|^2
    - int_Y V(U,eps)
    + S_W[s,A,U]
```

with:

```text
U = Ad(eps^-1) beta,
theta = A - Gamma(eps) - U.
```

A first-order parent action with momentum `P` is equivalent after eliminating `P`:

```text
S_IG-dyn =
  int_Y <P, D_A U>
  - 1/(2Z_U) int_Y |P|^2
  - int_Y V(U,eps)
  - c_theta/2 int_Y |theta|^2.
```

### EL Consequences

The second-order branch gives schematically:

```text
E_A:
  g_A^-2 D_A^* F_A - c_theta theta + J_U = 0.

E_U:
  c_theta theta - Z_U D_A^* D_A U - V_U = 0.

E_eps:
  c_theta L_eps^* theta + Z_U K_eps(D_A U) + V_eps = 0.
```

The first-order branch makes the same point with the canonical current:

```text
E_P:
  D_A U - Z_U^-1 P = 0.

E_U:
  -D_A^* P + V_U + c_theta theta = 0.

E_A:
  g_A^-2 D_A^* F_A - c_theta theta + [U,P] = 0.
```

Nonzero theta is allowed because the IG equation is differential, not an algebraic
`theta = 0` constraint. But the advertised source equation is generically replaced
by a total-current law:

```text
D_A^* F_A = theta_eff,

theta_eff = g_A^2(c_theta theta - J_IG),
```

where `J_IG` is `J_U` or `[U,P]` plus any eps-dependent current terms. The conserved
quantity is the total source `theta_eff`, not theta alone:

```text
D_A^* theta_eff = 0       from the full gauge Noether identity,
```

while `D_A^* theta = 0` is not generic.

### Exact Schwarzschild/Kerr

Branch 3 is the only branch with enough new dynamics to plausibly cancel the exact
Willmore residual without freezing IG variables. But it does not pass the exact gate
until the written action fixes `Z_U`, `V`, all current terms, and the section variation.

The exact black-hole proof obligation is:

```text
Find (A,U,eps,beta) for Schwarzschild and Kerr such that
E_s = E_A = E_U = E_eps = 0
```

with `Psi = 0`, finite action, and no hidden matter stress. A cancellation by
`J_IG` must be a vacuum GU current, not a relabeling of matter.

### FLRW `xi R theta^2`

Branch 3 is the strongest dark-energy branch because a covariant IG kinetic term can
naturally reduce to a scalar kinetic term:

```text
-1/2 (partial B)^2.
```

It is also the branch most likely to generate curvature couplings through reduction
of:

```text
|D_A U|^2,  |D_A theta|^2,  S_W[s,A,U],
```

and normal-bundle curvature terms. But "most likely" is not a coefficient. The branch
must compute:

```text
Z_theta,
C_Rtheta,
xi_eff = C_Rtheta / Z_theta.
```

Without a fixed `S_IG-dyn`, `xi_eff` is an adjustable model parameter and cannot be
counted as a GU prediction. Therefore Branch 3 does not derive `xi_eff < -0.319`
at current evidence.

### Outcome

```text
nonzero theta:        PASS
source equation:      REVISE to D_A^*F_A = theta_eff
exact Schwarzschild:  OPEN/FAIL until full EL solution is shown
exact Kerr:           OPEN/FAIL until full EL solution is shown
FLRW xi gate:         FAIL/UNPROVED; strongest route but no coefficient derived
```

Branch 3 is physically honest, but it forces a canon change: the source is a total IG
current, not bare theta.

## Branch 4: Bare Free-Beta Norm

### Definition

Take the bare theta norm with freely varied beta:

```text
S = S_YM[A] - c_theta/2 int_Y |A - Gamma(eps) - Ad(eps^-1) beta|^2 + S_W.
```

No constraints, no background declaration, no IG kinetic term.

### EL Consequences

The variations are:

```text
E_A:
  g_A^-2 D_A^* F_A - c_theta theta = 0.

E_beta:
  c_theta Ad(eps) theta = 0.
```

Thus:

```text
theta = 0,
D_A^* F_A = 0.
```

The desired nonzero dark-energy source is gone. The branch is not rescued by changing
the sign of `S_theta`; the beta equation still sets theta to zero.

### Exact Schwarzschild/Kerr

With theta killed, this branch collapses toward the Willmore-only section gate plus
ordinary Yang-Mills vacuum. It does not cancel the exact Schwarzschild/Kerr residual
and does not support the theta source needed by the weak-field/YM lane.

### FLRW `xi R theta^2`

There is no nonzero theta scalar mode, so the FLRW theta dark-energy analysis is
inapplicable. A bare inserted `R theta^2` term would still vanish on the beta equation
unless the variation status is changed, which means it is no longer this branch.

### Outcome

```text
nonzero theta:        FAIL
source equation:      FAIL; collapses to D_A^*F_A = 0
exact Schwarzschild:  FAIL at current evidence
exact Kerr:           FAIL at current evidence
FLRW xi gate:         FAIL; theta mode killed
```

This branch is structurally rejected.

## Can Any Branch Derive `xi_eff < -0.319`?

No.

Current evidence supports only this conditional:

```text
If the reduced GU action generates xi_eff < -0.319,
then the FLRW theta mode can reach the DESI-sign window.
```

It does not support:

```text
The GU action generates xi_eff < -0.319.
```

Branch-by-branch:

- Branch 1 has no IG dynamics and no computed curvature coefficient.
- Branch 2 preserves the source equation but leaves the constraint and reduction undefined.
- Branch 3 can naturally host the needed calculation, but without a fixed `S_IG-dyn` the
  coefficient is free.
- Branch 4 kills theta.

A bare term

```text
S_Rtheta = -1/2 xi int_X sqrt(-g) R[g] |s^*theta|^2
```

would decide the gate only by insertion unless a primary GU action fixes it, including
sign and normalization. A tunable `xi` is phenomenology, not a GU derivation.

## Hegelian Closure

### Steelman

The strongest steelman is that IG variables are not ordinary free auxiliary fields.
They are Stueckelberg/geometric coordinates on a tau-plus/double-coset slice. On this
reading, the allowed variations are constrained before the action is varied, so the
beta equation projects out only admissible directions and does not force `theta = 0`.
The connection equation can remain:

```text
D_A^*F_A = theta.
```

This steelman preserves the existing weak-field Schwarzschild lane and the canon
theta-source intuition with minimal surgery.

### Antithesis

The free-beta calculation is merciless:

```text
delta_beta S_theta = 0  =>  theta = 0.
```

So the action cannot keep both of these without extra structure:

```text
beta freely varied,
nonzero theta source.
```

Even after escaping that collapse, two independent physics gates still block promotion:
exact Schwarzschild/Kerr are not known full solutions, and the required negative
`xi_eff` has not been derived.

### Synthesis

Run Branch 2A first as the exact branch:

```text
Phi(eps,beta,s)=0,
D_A Phi = 0,
beta not free,
D_A^*F_A = theta.
```

The branch lives or dies on whether `Phi` is forced by GU geometry rather than invented
to save theta. If `Phi` cannot be sourced, switch to Branch 3 and explicitly rewrite
the source equation as a total-current law:

```text
D_A^*F_A = theta_eff,
theta_eff = c_theta theta - J_IG.
```

### Closure Conditions

The action/IG lane can upgrade only when all of the following are supplied in one
consistent branch:

1. A written action or primary IG geometry fixes the variation status of `(eps,beta)`.
2. The full EL tuple is derived:

   ```text
   (E_s, E_A, E_eps, E_beta, E_Psi) = 0.
   ```

3. The source law is stated exactly:

   ```text
   D_A^*F_A = theta
   ```

   or the corrected total-current equation is used everywhere.

4. Exact Schwarzschild and Kerr are solved as vacuum configurations of the full EL
   system, or the lane accepts a modified-gravity/strong-field failure.
5. The FLRW reduction computes `Z_theta`, `C_Rtheta`, and

   ```text
   xi_eff = C_Rtheta / Z_theta
   ```

   with `xi_eff < -0.319` generated by geometry, not inserted.

6. The scalar-mode assumption for `s^*theta` survives. If the pullback is spin-2,
   the KG dark-energy window does not apply.

A-independent constrained IG / Stueckelberg geometry.
