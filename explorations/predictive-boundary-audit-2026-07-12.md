# Predictive Boundary Audit After Wave 17

Purpose: make the H34 boundary explicit. This is not a new derivation, not a claim-status change, and not a
canon promotion. It is a sorting pass over the current Wave 17 / one-residual picture: which statements are
actual predictions, which are existence or consistency clears, which are fitted/imported accommodations, and
which remain blocked on the unbuilt source action.

## Audit Verdict

The current program has a strong **compression result**, not yet a strong **prediction result**.

The compression result is real: after the Wave 17 terminal source-action pass, the open freedom is tightly
localized. Gauge-vacuum selection, gravity soldering plus scale, the fermion `C2`/generation residual, and the
source-action causal-cure term all route to one source-action declaration-and-coefficient problem. That is a
valuable structural map.

But the prediction boundary is stricter. A statement should count as a GU prediction only when the relevant
quantity is emitted before target comparison by source-side structure, with no target-fitted coefficient,
chosen carrier, chosen vacuum, or post-hoc branch selection doing the load-bearing work. On that standard,
most current positive statements are **existence / consistency / conditional-theorem** statements rather than
predictions.

## Clean Prediction Rule

Use this rule before calling any next result a prediction:

1. **Source-first emission:** the quantity must be produced from the source-side structure before comparison
   with the target fact.
2. **No target imports:** do not use `3`, `24`, `24/8`, `chi(K3)=24`, observed SM algebra, DESI-fit values, or
   known GR success as inputs to choose the result.
3. **Fixed degrees of freedom:** any carrier, vacuum, coefficient, scale, boundary condition, or initial
   condition must be fixed by the same source-side rule, not selected afterward because it works.
4. **Independent failure condition:** the result must name what would falsify or demote it.
5. **Grade stays local:** a prediction in one sector does not upgrade another sector by association.

## Current Genuine Discriminators

These are the nearest places where the program could move from compression/consistency toward prediction or
disproof:

- **H41: causal-cure term.** Build the non-minimal RS / field-space-defining term that cures the built VZ
  leakage without p-hacking the carrier. This is the terminal object. It decides whether the `{A,B}` fork
  remains a 1-bit postulate or becomes source-forced.
- **H10: weak-field with matter / PPN.** The gravity branch can be cheap-tested against real solar-system bars
  without waiting for a full source-action build. A failure here would be more decisive than another synthesis.
- **H26: loop-level ghost unitarity.** The tree-level Krein/Bateman-Turok clearance does not decide the generic
  Stelle-Mannheim loop problem. Surviving this would materially strengthen the gravity leg.
- **Dark-energy coefficient gate.** A source-derived `xi_eff`, `f_0`, or equivalent coefficient recorded before
  comparison would turn the current sign-consistency into a real predictive test. Without that, it remains a
  fitted/degenerate accommodation with a useful near-falsifier.

## Anti-Overclaim Ledger

- Do not call SM/forces "derived" while the sub-block/vacuum selection is source-action-gated.
- Do not call dark energy "cleared" while the EOS machinery is reconstruction-grade and amplitude-degenerate.
- Do not call gravity "resolved" while soldering, `mu_DW`, and loop unitarity remain open.
- Do not call the generation count "three" while the current honest set is `{1,3}`.
- Do not call the one-residual thesis a prediction. It is a compression of residual freedom into one named
  source-action problem.

## Net

The Wave 17 picture is stronger than a loose list of open problems: it identifies one terminal construction
whose outcome would propagate across sectors. The honest public boundary is equally important: until that
source-action object emits fixed selections or coefficients before target comparison, the program's best
positive results remain mostly existence, consistency, conditional theorem, and compression results.

That is still progress. It tells the next run exactly what must change epistemically: build or falsify the
causal-cure source-action term, run the cheap weak-field and loop discriminators, and refuse to upgrade a fit
into a prediction merely because the one-residual map is elegant.
