## 1. The schema instance (what was built)

**Site/cover (A1).** `U = {O1, O2, O3}`: three observer regions = three modular-weight profiles
(w = 1.0 / 0.45 / 0.20) on the shared `W98` tower -- the same constants as `W110`. The nerve has
3 opens, 3 pairwise overlaps, and the triple overlap: a **genuine cover cycle**, honoring T226's
structural correction (single-overlap "monodromy" is a CSP artifact, not an H1 class).

**F_class (A2)** -- coefficient group **(R, +), declared** (T59 gate G1): 0-cochains = per-region
potentials; 1-cochains = the **transition data** `theta_ij(k) = log((1-r^i_k)/(1-r^j_k))` (W103's
regularized relative tail-phase), **carried, not projected away** (gate G2). Computed: the cocycle
`theta_01 + theta_12 + theta_20 = 0` to `< 1e-12` at every k in `1e2..1e6`, and the 1-cochain is
**exhibited as a coboundary** `theta_ij = F_i - F_j` with `F_i = -log w_i`. In T226's language:
`[g] = 0`, **H1(U, F_class) = 0, a global section exists** -- and its non-phase half glues too
(two radically different global weights give the same `2[P]` to `< 5e-3` and the same Krein-null
line to `1 - 1e-9`; W103 rate-independence doing the work).

**F_op (A4)** -- operator coefficients: sections over `O_i` are the bounded conjugation candidates
`eta_i^{-1/2}`. **Descent fails**: the union's sup-conditioning diverges under UV doubling
(`1779 -> 3557 -> 7112` -- W110 T4's exact figures reproduced), every region's own diverges, and
every pairwise operator disagreement diverges (`> 10` at `k = 1e5`) while the class differences are
compact. No global operator section exists at any level, on the **same cover** where F_class glues.

**The morphism (A5).** `q : F_op -> F_class` is the class/quotient map. Checked: the transition
data extracted **from the operator sections** (log-ratio of the min-eigenvalue gaps of
`eta_i, eta_j` on each overlap) equals the direct class transition data to `< 1e-10` -- the
level-split is one commuting diagram, not two disconnected computations.

**Falsifiability of the machinery (A3, A6).** The cocycle check **fires on a planted bad triple**
(T226's own validation requirement). And T59's false-section trap is closed **on GU's own data**:
in the non-coplanar (U5-violating) control, the coefficient-**blind** scalar encoding reports a
global section (all blind transition data identically zero -- a **false section**, the Mobius-trap
shape), while the coefficient-**aware** encoding carries the frame aligners and finds
pairwise-exact data whose loop composition is the **0.403-rad holonomy** -- no global frame
section, and pairwise class coherence broken non-compactly. The adapter's H1 verdicts can fail,
and demonstrably do on the control.
