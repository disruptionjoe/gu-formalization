# Conclusion of the gate pipeline

## What we set out to test

Joe's live/dark observer-sheaf hypothesis: the one W211 bit (`bar(b)`, the Krein grading sign) is a global
class carried by the collective observer nerve, sourced by per-observer record issuance, universal, and
robust to stalk churn. The four-gate falsifier (reordered by the ten- and twenty-persona panels) tested it.

## What the gates returned

- **Gate 0/1 (b1, assumption-light):** PASS. The issued observer nerve carries loops that are generic,
  extensive, and deletion-stable across ER/BA/RGG models. This cleared the `S^3` "no non-vacuous home" wall
  for the H1 / pi_1-holonomy reading.
- **Gate 2a (exact 𝔽₂ / XOR-SAT):** CONDITIONAL PASS with a sharp condition. Universality (a unique global
  sign, no domains) holds **iff the sign is vertex-sourced** — a per-observer coboundary (`[J]=0`). In that
  case it is SAT in every model and **churn-stable** (20/20 after 20% deletion). Any relational (edge-sourced)
  component frustrates into domains -> falsified.
- **Gate 2b (dynamics):** NOT NEEDED. The passing branch is decisively unique (no degeneracy for a dynamics
  to break).

## The honest walk-back (stated plainly)

The passing branch is the vertex-sourced one, and **a vertex-sourced coboundary has trivial holonomy on
every cycle.** Therefore, in the H1 / pi_1 reading, **the nerve's loops carry no sign information** — the
`b1` we established in Gate 0/1 does **not** determine the value. It only had to be *consistent* with the
sign, not fix it.

So the romantic form of the hypothesis — "the observer nerve *derives* the generation sign as a loop
holonomy" — **does not survive in the H1 reading.** The dichotomy is sharper than "vertex good, edge bad":

- the branch where the topology *would* derive the value (relational, nontrivial holonomy) is the one that
  **fails** universality (domains);
- the branch that **succeeds** (vertex/alive) makes the topology **irrelevant to the value**, leaving a
  single external bit.

This is a genuine, if deflationary, finding, and it is stated rather than smoothed over.

## What genuinely survives (and it is worth having)

A **conditional structural result** that **sharpens W211**:

> If the Krein sign is a per-observer (alive / issuing) datum, then it is **universal, domain-free, and
> churn-stable**, and `bar(b)` reduces to **exactly one external bit** — the orientation of the
> issuance/finality arrow. If it is relational, it frustrates into domains.

W211 already reduced the bit to one external ℤ/2. The new content is: (i) it **explains why exactly one**
(vertex-sourcing kills all the loop-holonomies, leaving a single global flip), and (ii) it **proves there are
no domains** in the passing branch. That is a real advance on the structure of the bit — a reduction, not a
derivation of its value.

## The one place a topological derivation could still live (not closed)

Personas 2/6 (obstruction theory / higher category) note H1 is the **primary** obstruction; if it vanishes
(as it does in the vertex-sourced branch), the **H3** 2-gerbe class becomes the **secondary** one and *could*
be nontrivial even when H1 is trivial. H3 was **not** tested (it needs the full Čech nerve with triple+
overlaps). So the honest status: the H1 branch reduces the value to one external bit; **H3 is the only
untested place a topological derivation of the value could still survive.** It is a live parallel thread, not
a closed door.

## The one remaining question, and where it lives

Given the H1 reduction, the value of the single remaining bit is **the orientation of the issuance/finality
arrow** — a **temporal-issuance** datum. This is W211 **path (a)** (source the bit from TI/TaF), and it is
exactly Joe's original chain: issuance sources the sign. The precise cross-repo question (Joe-gated) is drafted
in `ti-question-finality-orientation-as-krein-sign-2026-07-15.md`.

## Standing walls (unchanged)

The generation **number** stays behind the unwritten source action. Krein **positivity** stays an external
posit. The cross-repo identity (Krein sign = finality orientation) is **Joe-gated**. Nothing here is promoted.
