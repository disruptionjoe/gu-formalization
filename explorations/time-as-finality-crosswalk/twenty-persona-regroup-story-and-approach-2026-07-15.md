# Twenty-persona regroup

## The canonical story (one paragraph)

A single undecided bit in a physics theory — the Krein sign `bar(b)` that decides whether Geometric Unity
forces exactly three particle generations — was chased to ground. Across three independently-built formal
repos it turned out to be the **same ℤ/2 signed-graph object**. Its **consistency** (all observers agree; no
fragmentation into domains) is **provably one object** (Harary balance; 720/720 across three independent
algorithms). Its **value** is a single **leftover bit = the orientation of a finality arrow**, which the
undirected consistency object **structurally cannot fix** — which is exactly why GU cannot decide it alone,
and why the labour splits: **geometry supplies consistency, finality supplies orientation.** The last open
question is whether that finality direction is **derived** or **posited**.

## The twenty (story in their idiom / best approach to find the truth)

1. **Sheaf/topos** — *Story:* local signs glue (a coboundary); the leftover is one global section's orientation.
   *Approach:* build the second adapter as a functor between the **directed** sites; the identity is a descent statement.
2. **Obstruction theory** — *Story:* the H1 obstruction vanishes; the value is an H0-level global choice, the arrow the missing datum.
   *Approach:* compute the **relative** obstruction between the undirected and directed objects.
3. **Spin geometer** — *Story:* the sign is an orientation/spin-structure; consistency fixes all but the global orientation.
   *Approach:* treat the finality arrow as the orientation form; ask if it is canonical.
4. **Discrete Hodge** — *Story:* harmonic ℤ/2 space is trivial (balanced); the free bit is H0; direction is a 1-form the balance metric ignores.
   *Approach:* add the **directed/magnetic** Laplacian; the arrow is its kernel phase.
5. **Lattice ℤ/2 gauge** — *Story:* the gauge field is pure-gauge; the residual is a global holonomy the undirected theory can't see.
   *Approach:* add a **directed link variable**; the value is a Polyakov-line orientation.
6. **Higher-cat/HoTT** — *Story:* one truncation (balance) is settled; the value lives one level up — a `B(ℤ/2)` torsor with no basepoint.
   *Approach:* the second adapter must **provide the basepoint** (the arrow).
7. **Holographic code** — *Story:* consistency = the code is well-defined; the value = a logical bit no boundary region can read.
   *Approach:* test whether the finality arrow is the **logical-Z** operator that fixes it.
8. **Percolation** — *Story:* below frustration threshold everywhere; the leftover is a global order parameter with two symmetric values.
   *Approach:* find the symmetry-breaking field (the arrow) and whether it percolates.
9. **Network-coding / max-flow** — *Story:* a consistent global assignment is feasible; its sign is free unless a directed capacity biases it.
   *Approach:* model the arrow as a **directed capacity**; does it force the flow's sign?
10. **Spectral graph** — *Story:* the signed Laplacian is balanced (zero frustration eigenvalue); the free bit is the Fiedler sign.
    *Approach:* a **magnetic Laplacian** encodes the arrow; check whether its phase fixes the sign.
11. **Byzantine agreement** — *Story:* the network agrees (consistent), but the agreed value is convention unless an oracle seeds it.
    *Approach:* is the finality arrow a valid **unforgeable seed** (a genesis)?
12. **Threshold secret-sharing** — *Story:* the sign is a secret everyone reconstructs up to one flip; the arrow is the missing share.
    *Approach:* model finality as the **(k,n) dealer**; is the dealer canonical?
13. **XOR-SAT** — *Story:* the system is SAT with a 1-dimensional solution affine space; the arrow picks the coset representative.
    *Approach:* show the arrow is a **linear functional** selecting the representative.
14. **Metastable consensus** — *Story:* consensus is reached (no domains); which value is the metastable basin is undetermined without a bias.
    *Approach:* is the finality **reversal-cost asymmetry** the tie-breaking bias?
15. **CRDT / confluence** — *Story:* merges are confluent -> consistent; the initial value is the one free seed.
    *Approach:* is finality a **monotone / lattice-bottom** that canonicalizes the seed?
16. **Nakamoto** — *Story:* no forks (consistent chain), but genesis orientation is chosen.
    *Approach:* does finality supply a **canonical genesis** (longest-cost-to-reverse)?
17. **Gossip / epidemic** — *Story:* the bit homogenizes (consistent); its value is set by whoever seeds first, absent a gradient.
    *Approach:* is the finality **gradient** (reversal cost) that global seed?
18. **Spin glass** — *Story:* it's a **ferromagnet, not a glass** (unfrustrated) — unique up to a global flip fixed only by an external field.
    *Approach:* the finality arrow is the external field `h`; measure whether `h != 0` is **derivable**.
19. **Topological order** — *Story:* trivial topological sector; the flip is a superselection label set by the boundary.
    *Approach:* the arrow is the **boundary condition**; is it canonical?
20. **Krein QFT (honesty anchor)** — *Story:* consistency = the indefinite metric is gauge-fixable; the value = which sign is positive-norm, an external posit.
    *Approach:* the arrow can at most give "**which sign given positivity**," never positivity itself — keep that truth-ceiling explicit.

## Chairman synthesis

**The story they converge on** is the one-paragraph account above, and remarkably ~15 of the 20 name the
*same shape* independently: **a settled consistency layer + a single free global bit that only an external
directed datum (genesis / oracle / external field `h` / basepoint / boundary condition / orientation form)
can fix.** Ferromagnet-not-glass (18), torsor-without-basepoint (6), SAT-with-1-dim-affine-solution (13),
genesis-block (11/16), external-field-`h` (18) are all the **same** statement: consistency is done; the value
is one un-basepointed ℤ/2 that a directed structure must seed.

**Best approach to maximize understanding and find the truth (panel consensus):**

1. **Second adapter contract on the DIRECTED structures.** Map TI `Ext_S` (directed morphisms) <-> TaF T18
   (finality arrow) and test whether *they* are one object — the orientation lives here, not in the undirected
   balance object. This earns the >=2 adapter contracts the tri-repo rule requires before any identity, and it
   targets exactly where the value lives. (Personas 1,2,4,5,9,10 — the "directed/magnetic" upgrade.)
2. **The is-the-arrow-canonical test.** Every persona's "approach" reduces to one question: is the directed
   datum (genesis / dealer / external field / boundary) **canonical/derivable**, or a **free posit**? That is
   TaF's T18-vs-T57 question. It is the single decisive experiment for the **value**.
3. **Honest truth-ceiling (persona 20).** Even a positive answer yields "the value = a canonical finality
   direction **if one exists**, else a posit" — it never derives Krein *positivity* itself. State that ceiling
   on every result.

## Where the work should live (decision)

**possibility-to-capability (P2C).** Its ratified charter is the neutral cross-repo layer for exactly this
(possibility -> ... -> the open question of finality), with `interfaces/` for **adapter contracts** and
`synthesis/` for **repo-independent story**. The convergence work is repo-independent by nature; it belongs
there, imported as **Frozen-Packet v0.2** honouring P2C boundaries (no editing source truth; each verdict
sovereign; preserve honest bounds; no artificial joint success). **GU / TI / TaF keep their sovereign
artifacts and have received awareness mailbox notes.** GU remains the home of the *consistency* leg (Gate
2a); TI/TaF own the *orientation* leg; P2C hosts the *convergence/synthesis*.
