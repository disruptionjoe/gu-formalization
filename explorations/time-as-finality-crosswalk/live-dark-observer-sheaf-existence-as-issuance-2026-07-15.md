# The live/dark observer sheaf

**Attack this note; do not defend it.** It is a synthesis hypothesis. Its job is to state one
possibly-new lever precisely enough to be killed, and to inherit — not launder — the walls the two
threads it combines already hit.

## 0. The claim in one paragraph

Each existing observer is a **live stalk** of a sheaf of local Standard-Model-carrying geometries over
an **observer site**. "Existing / being on" is not a separate fact from "issuing records": to be a live
stalk *is* to issue records into the total structure (this is the `record-issuance-boundary` thread). The
generation-count sign (`bar(b)`, the one W211 bit) is not a datum of any single stalk; it is a **global
cohomology class** (a holonomy / Čech class) of how the live stalks **glue** — which is why it is (a)
universal across observers and (b) invisible to any one of them. When an observer turns off, its stalk goes
**dark**: it drops from the support, but the regional gluing among the remaining live stalks and the global
class are **unchanged**. The candidate NEW move is *where the holonomy lives*: not on the spatial boundary
`S^3` (simply connected, so vacuous — the wall the 2026-06-26 thread hit), but on the **collective
observer-overlap nerve**, whose fundamental group / H3 can be non-trivial precisely because many observers
glue in a network with loops.

## 2. The candidate NEW lever, stated so it can be killed

**Lever.** Relocate the APS boundary holonomy from `pi_1(spatial boundary) = pi_1(S^3) = 1` to
`pi_1` (or the relevant H3) of the **live-observer overlap nerve** `N_live`. If `N_live` is generically
**non-simply-connected**, then `rho: pi_1(N_live) -> U(16)` (or the degree-3 Čech class E054 computes) can
be **non-trivial without stipulating a lens space** — the missing topology is supplied by the *collective
structure of many observers gluing*, which is exactly the thing record-issuance dynamically builds.

**Why it is not obviously empty.** E054 already returns holonomy **−1** on an H3 Čech **record-graph**
fixture from C-operator parity. A record-graph is a nerve. So there already exists a computed non-trivial
class on a record-nerve; the lever is the claim that this is the *generic* situation for `N_live`, not a
one-off fixture.

**The three obligations that would make it real (each is a falsifier if it goes the wrong way):**

1. **Non-simple-connectedness must be FORCED, not chosen.** Show `N_live` is generically
   non-simply-connected *from the issuance/gluing dynamics*, not by hand-picking a nerve with a loop. If the
   generic live nerve is contractible (or its class is stipulable either way), `rho` is trivial/free and the
   sign is *not* forced — **dead**, and it dies the same way the `S^3` version died. This is the
   already-tracked H3 obligation "cocycle values forced rather than stipulated" (E054's are stipulated),
   relocated to the nerve.

2. **Robustness to stalk deletion must be PROVEN, not assumed.** Joe's "turn one observer off and the
   regional/global persist" is the claim that the global class is **invariant under support shrinkage** down
   to some minimal live nerve. Plausible (Čech classes restrict), but "restricts" is not "unchanged": one
   must show the class is *carried* by the minimal live nerve, i.e. deleting a stalk cannot trivialize it.
   If a single deletion *can* flip/kill the class, the count would be observer-fragile — **falsified on
   sight** (one death does not change the number of generations).

3. **"Collective RECORDS" vs "discardable redundancy" must be distinguished.** T12' proved zero-trace but
   **explicitly did not** separate "collective records" from the standard ghost reading "discardable
   redundancy" — both have identical zero individual trace. The sheaf picture needs the collective class to
   be *record content* (provenance-bearing), not gauge redundancy. This adjudication is behind the dynamics
   / the boundary adapter, same as T12' leg #1.
