# Negative Finding: Signed-Readout Monotonicity Criterion Is Rate-Independent

**Status:** Exploration-grade negative finding. Not a candidate for canon. Registers a clean null result from the five-run issuance-rate / observer-contact analysis (2026-06-22).

**Source:** Five-run synthesis at `five-run-issuance-rate-observer-contact-2026-06-22.md`, cross-run convergence item 4.

---

## The Negative Finding

All five runs of the issuance-rate / observer-contact analysis agreed on the following:

> For all values of `lambda*(s)` in TaF's admissible range, the signed-readout monotonicity criterion — GU's current main active-research theorem — is unchanged in statement, scope, and failure conditions.

This is a clean negative result: the issuance rate concept adds no formal content to GU's primary theorem lane.

---

## Precise Statement

Let `lambda` be TaF's optimal issuance rate (a positive real parameter) and let `lambda*(s)` be the value of this rate at information state `s`.

Let the signed-readout monotonicity criterion be as stated in `active-research/signed-readout/signed-readout-verdict-2026-05-31.md`:

**Criterion (Signed Readout Monotonicity).** Let `E` be the free commutative monoid generated by local contribution events with information order `e <= e'` iff `e' = e + d` for some `d in E`. Let `G` be an ordered abelian group with positive cone `G_+`. Let `R : E -> G` be the scalar readout. Then `R` is monotone iff every generator has nonnegative weight in `G_+`.

The criterion makes no reference to:
- the rate at which generators are added to `E`,
- the rate at which the observer assembles its record of elements of `E`,
- any cadence or throughput parameter in the source-to-shadow chain.

The criterion's statement is purely structural: it concerns the weight assignment on generators, not the dynamics of accumulation.

**Negative result:** Substituting any finite value of `lambda*(s)` into the criterion produces the same theorem. The criterion has no lambda-dependence.

---

## What This Rules Out

This negative finding rules out the following:

1. TaF's issuance rate modifying the monotonicity condition for GU's signed-readout theorem.
2. Lambda*(s) appearing as a parameter in the iff condition for monotone provenance.
3. The issuance rate concept providing a new failure mode for the signed-readout criterion beyond those already identified (negative-weight generators).
4. The issuance rate providing a route by which the criterion could be satisfied at some rates and violated at others.

---

## What This Does Not Rule Out

This finding is narrow. It does not rule out:

1. The possibility that `lambda_max` (the saturation threshold) is a meaningful parameter in GU's L4 or L2 axes under a different formalization.
2. The BvN classicality threshold `Gamma_min` coinciding with `lambda_max` in a non-trivial way (Run 3 hypothesis — separately unconfirmed).
3. The filtered-sheaf temporal obstruction (Run 4 concept) as a distinct formal object in GU's exploration lane.
4. The issuance rate belonging to L6 (coordination loop dynamics) as an unspecified parameter that does not currently constrain any GU result but might do so under future development.

## No-Go Class Relative Map Contact

The no-go class relative map (`canon/no-go-class-relative-map.md`) specifies four no-go families: Witten 1981, Nielsen-Ninomiya, Freed-Hopkins, and Distler-Garibaldi. The rate-independence result extends naturally:

For all four no-go families in the map, the class assumptions (smooth manifold, locality, invertible EFT, single E8 representation) are structural properties. None of them are rate-indexed. The rate at which an observer builds its record of the substrate has no purchase on whether the substrate satisfies the class assumptions. Therefore the negative finding generalizes: the issuance rate has no formal contact with any of the four no-go families via their class assumptions or known class exits.
