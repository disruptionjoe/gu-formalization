## 2. Why the no-go is still class-relative: the consolidated case

Across all eighteen lenses plus synthesis, four convergent facts establish class-relativity:

1. **DG computes a compact-group invariant.** The reality type FS(V) is a Haar integral over g^2, a finite integer read off a character table, defined only because G is compact. The eta-invariant is a transcendental spectral quantity mod Z. The two provably coincide on compact internal groups and diverge exactly when the centralizer is non-compact (SO(5,5) inside Spin(7,7)), which is GU's regime. The obstruction therefore lives in the target of a forgetful functor that retains only the compact reality type, not in the richer pre-image.

2. **An explicit inside-the-single-group shadow exists.** phi_Hilbert deletes the indefinite signature and the ghost parity and reproduces DG's null answer on its image, unlike every published escape (E8xE8, SO(3,11), K(E10)), each of which is a category change of the gauge group. This is the first demonstrated metric-forgetting shadow, not merely an asserted one.

3. **The even-ness backbone is a compact/closed theorem.** Rokhlin (signature congruent to 0 mod 16), the quaternionic Kramers wall (J^2 = -1), and the real-adjoint 4k/12k/24k counts are all theorems about closed spin manifolds and the interior Atiyah-Singer integrand. On a manifold with boundary the index acquires a real-valued APS term (eta + h)/2 that none of these constrain.

4. **The published escapes are category changes precisely because the projection is non-definable.** The PO1 checker classifies DG as AC3-fail / boundary_non_definable: GU does not project onto single-E8, it scope-exits to a different site (Sp(64), Lorentz not inside the gauge factor, chirality as ind_H on the non-compact Y14). When the projection is non-definable, category change is the only available exit, so reading those exits as "cheating" mistakes a diagnostic signature for a violated rule.

**Critic's correction (the load-bearing honesty caveat the panel blurred).** The meta-analysis celebrates "positivity = compactness = one axis" as a discovery, but the identity cuts against the Krein cluster's self-description. A finite-dimensional representation of a compact group always admits an invariant positive-definite Hermitian form (Weyl unitary trick). The Krein form K = eta_V (x) beta_S is indefinite only because the internal G was taken non-compact (SO(5,5), not SO(10)). DG assumption (3) explicitly requires G compact. Therefore dropping positivity is not an independent "forgotten datum" layered on DG's setup; it requires already having left DG assumption (3). The Krein hatch preserves "single finite-dimensional group" but does not preserve "compact centralizer," so it is inside the broad single-group class yet outside DG's actual hypotheses. The panel conflated these two. Honest restatement: the Krein datum is a genuine scope-exit (non-intersection), and labeling it "the first inside-class richer datum" is partly motivated cope, because admitting it is already the compactness drop. This does not weaken the class-relativity conclusion (it strengthens the non-intersection reading), but it removes the Krein hatch from the "inside-class enrichment" column and files it with the scope-exits.
