# W115: TI's E043 discriminator, run on GU's strip-cost functional

**The question.** TI's central missing object is the issuance measure `mu` (signature axiom:
SAX subadditivity, E043). GU has a computed functional that prices selection-finalization: the
strip cost -- the Connes-cocycle continuation to `-i/2` has per-mode norm
`sqrt(cond(eta(r_k)))` exactly (W109 adversary-(b) computation; the W106 KMS/crossing
bottleneck; the W113 Petermann dictionary). Does the strip cost satisfy TI's mu axioms?
Run TI's OWN discriminator on it, both sides, kill-honest.

**Answer: NOT-A-MU-CANDIDATE.** The discriminator classifies the strip cost into the
**additive family** (E036/E043's action/linear-information column) -- the one class E043
explicitly rules out as a carrier of the subadditivity claim -- and the D-FORK analysis
places it on TI's **cost-of-finality** route (E140), not the issuance side.

---

## 1. E043's axioms and test protocol, exactly as TI records them

From `E043` Section 6 (**SAX**, the formalized TI-C021):

- **(SAX-0)** `mu(empty) = 0`;
- **(SAX-1, monotone)** `A subset B => mu(A) <= mu(B)`;
- **(SAX-2, strict size-subadditivity)** for disjoint `A, B` with `|A|,|B| > 0`:
  `mu(A u B) < mu(A) + mu(B)`;
- **(SAX-3, size-determined)** `mu(A) = g(|A|)` for a strictly concave `g` with `g(0)=0`
  (canonical realization `g(s) = c s^alpha`, `0 < alpha < 1`).

The **entropy side** (`P_corr`, E043 Section 2): `H(A,B) <= H(A) + H(B)` with equality iff
`A` and `B` are independent; the deficit is exactly the mutual information `I(A;B)`.

The **discriminator's test protocol** (E043 Sections 3-4, Theorem 4 both directions):

1. **Direction 1 (disjoint independent inputs):** on disjoint *independent* `A, B`, the
   entropy deficit is identically **0** while genuine `mu_alpha`'s deficit is strictly
   **> 0** (the concavity gap `c(|A|^a + |B|^a - (|A|+|B|)^a)`). A functional's column is
   read off from where its deficit lands on this class.
2. **Direction 2 (correlated inputs, additive baseline):** correlation produces a strict
   entropy deficit that an **additive** measure (`alpha = 1`) does not register at all --
   the additive family is the **null class for both predicates**.

E043 Section 1 additionally records: cardinality, action, probability, and linear
information are additive, so "none is subadditive, so none can absorb a subadditivity
claim." Section 5's category check: `mu` lives on **realized structure size**
(distribution-free); entropy lives on the **law of a random object** (distribution-bound).
E043 Section 6's regime caveat: SAX-2/3 are the **FTS-regime** signature; in the Godelian
regime `mu` may be linear-or-non-concave (sustained marginal issuance). E045's D-FORK makes
that fork the program pivot.

## 2. The strip-cost functional, typed

The GU object (W98/W109 instance: `m1=0, m2=0.3, G=0.1`; vertex growth `g_k = G k^p`):

```
r_k = g_k / (g_k + Dw(k)/2),   Dw(k) = |m1^2 - m2^2| / (w1 + w2)  ->  0 (UV)
cost_strip(k) = per-mode norm of the -i/2 cocycle continuation
             = sqrt(cond(eta(r_k))) = sqrt((1+r_k)/(1-r_k))  ~  k^{(p+1)/2}
```

Typings tested as set-functions on mode-subsets `S` of the tower (the typing is half the
work; all four honest candidates were run, none chosen to force an outcome):

| Typing | Definition | What it is physically |
|---|---|---|
| **T-A** per-state strip form | `mu_f(S) = sum_{k in S} |f(k)|^2 cost_strip(k)` | the cost of finalizing (KMS/crossing-continuing) the selection carried by state `f` over the mode-subset `S` -- the honest "price of finalizing a selection over S" |
| **T-B** Araki leg | `sum_{k in S} |f(k)|^2 log cond(eta(r_k))` | the relative-entropy (log Delta_rel) cost, W106 nesting row 2 |
| **T-C** sup/uniformity | `N(S) = sup_{k in S} cost_strip(k)` | the norm of `Delta^{-1/2}` restricted to `S` -- the W98 break object itself |
| **T-D** trajectory (steelman) | initial segments `S_K`, size `K`, `g(K) = N(S_K)` | the most mu-favorable reading possible (answers the gerrymander adversary from the OTHER side: chosen to make it pass if anything can) |

## 3. Running the discriminator (tests/W115_e043_strip_cost.py, 20/20, exit 0)

**SAX side.**

- **T-A / T-B fail SAX-2 with exact equality.** A per-mode sum over disjoint index sets is
  additive by construction (D1: definition of a sum; D2: worst relative deficit `2.2e-16`
  over 50 seeded disjoint pairs x 2 states x 2 typings, and an 18-point
  `(G, m2, state)` sweep, all `< 1e-12`). Zero deficit means **not strictly subadditive**:
  the strip form is a *measure* in `k` (countably additive), the very family E043 Section 1
  excludes as a subadditivity carrier.
- **Every typing fails SAX-3.** Two disjoint 40-mode subsets (IR block vs UV block) give
  `mu_f` values differing by 56 orders of magnitude (T3: `1.77` vs `7.2e-57` on a Schwartz
  packet) and sup values differing by 10x (`18.9` vs `190.8`). The strip cost depends on
  **which** modes (UV vs IR identity) and **which** state -- never on the count `|S|`.
  There is no `g` with `mu(S) = g(|S|)`. The synthetic `mu_alpha` control passes the same
  test, so the test can tell.
- **T-C is strictly subadditive but for the wrong reason.** `max(a,b) < a+b` holds for
  positive entries (T4, 50/50 pairs), but the deficit is **exactly `min(N(A), N(B))`** --
  a dominance identity, not the concavity gap -- and identical-size subset pairs give
  deficits `16.4` vs `95.6`, so the deficit is not a function of `(|A|,|B|)`. Strict
  subadditivity with the wrong deficit source and no size-determination is not SAX
  (E043 Prop 3 characterizes SAX by strict concavity of the *size* functional).

**Entropy side (run, as the discriminator demands).**

- The W98 tower modes are statistically independent (per-mode block-diagonal metric,
  product state) -- so disjoint mode-subsets are exactly E043's **direction-1 class**.
  There: entropy deficit `= 0` (verified with per-mode occupation laws from the `p=0` pair
  tail `c_k ~ G/2k`; joint entropy additive to `< 1e-12`), **strip-cost deficit `= 0`**
  (same), genuine `mu_alpha` deficit `= 3.70 > 0` (T5). The strip cost sides with the
  **additive/entropy-null column**, lacking exactly the signature that defines `mu`.
- Is it entropy-type then? Only categorically: like entropy (and unlike `mu`), it is
  **state/distribution-dependent** (E043 Section 5's category axis -- it lives on the state,
  not on realized-structure size). But it is not `P_corr` either: it is a state-weighted
  **sum** of per-mode costs, so it can never produce a correlation deficit; it has no
  mutual-information term. The precise classification is **E036's additive family** --
  closest cousin: **action** (a state-weighted additive spectral cost). E043's direction-2
  case (Section 4) is exactly the additive baseline the strip cost realizes.

**The trajectory steelman (T-D) -- answering "you chose the domain to make it fail."**

The most favorable typing possible: size = UV depth `K`, `g(K) = N(S_K) ~ c K^{(p+1)/2}`.
Measured exponents `0.500 / 1.000 / 1.500` at `p = 0, 1, 2`:

- `p=0`: `g(K) = c sqrt(K)` -- strictly concave through the origin, literally E043's
  canonical `alpha = 1/2`. **SAX-shaped** -- but (i) initial segments are not closed under
  disjoint union, so SAX-2's "for all disjoint A,B" cannot even be *stated* on this domain
  (it is a one-parameter curve, not a measure), and (ii) it is not size-determined off the
  segments (T3).
- `p=1` (the physical derivative-coupled ghost, per W106's own fork discussion):
  exponent `1.000` -- **additive**, the null class.
- `p>=2`: superadditive, **increasing marginal cost** -- the *opposite sign* to SAX's
  diminishing marginal issuance (T6: per-octave increments `1526 -> 3053` at `p=1`).

So even the gerrymandered-in-favor typing is SAX-shaped only at `p=0`, and flips to
anti-SAX exactly at the physically honest vertex. The pass at `p=0` is a coincidence of
exponents (`(p+1)/2 = 1/2`), with the wrong deficit mechanism (spectral growth of per-mode
cost + sup absorption, not pool depletion), on a domain where the axiom is unstateable.

## 4. VERDICT and what it means

**VERDICT: NOT-A-MU-CANDIDATE -- the discriminator classifies GU's strip cost as a
state-weighted ADDITIVE spectral cost (action-type), living on TI's cost-of-finality route,
not the issuance side of D-FORK.**

Against the three pre-registered outcomes:

- **(i) SAX-passes: NO.** Fails SAX-2 (per-state: exact equality) and SAX-3 (every typing).
- **(ii) entropy-type: PARTIALLY -- category only.** It shares entropy's
  distribution-dependence (state-side of E043's category table) but not `P_corr`'s
  correlation deficit. The exact classification E043's own framework provides is the
  **additive family** (E036: cardinality/action/probability/linear-information) -- the
  null class of *both* predicates (E043 Section 4's baseline).
- **(iii) ill-typed: PARTIALLY -- against SAX-3 specifically.** SAX-3 demands
  size-determinedness (`mu(A) = g(|A|)`, distribution-free); the strip cost is
  irreducibly mode-identity- and state-dependent. That mismatch is not repairable by
  re-typing: it is what the functional *is* (a quadratic form of a multiplication
  operator, W106).

**The D-FORK question (the adversary's second press), answered.** TI's `mu` measures
issuance of **new** possibility (source-side, E045); GU's strip cost prices **finalizing
existing selections** over an already-realized mode tower -- consumption/commitment, not
issuance. TI already has a slot for exactly this object: the **cost-of-finality route
(E140)**, which TI keeps *separate* from the issuance measure and found largely absorbed
into standard thermodynamics (Landauer/Bennett family) except for agent-adversary sizing.
The strip cost slots into E140's **thermal/physics layer** (a per-mode, state-dependent,
additive commitment cost -- compare E140's "maintenance-of-irreversibility" classification),
not into `mu`. On D-FORK's axis the strip cost is regime-agnostic about `Theta_eff`: it says
nothing about whether the source's type space is finite or self-generating.

**What this stress-test does to TI's driving hypothesis (one-way rule: stress, not
support).** The cheapest cross-repo mu-bridge candidate is now **eliminated by TI's own
instrument**: if TI's three-layer hypothesis wants a GU face for `mu`, it cannot be the
strip cost. The informative content: (a) GU's selection-finalization cost is
**finality-priced, not issuance-priced** -- consistent with (but not evidence for) TI's
existing separation of those two routes; (b) any future GU-face candidate for `mu` must be
size-determined and distribution-free -- no per-state quadratic form on the mode tower can
qualify, which rules out the entire W106 nested hierarchy (metric, Araki, strip,
`Delta^{-1}`) in one stroke; (c) the one SAX-shaped shadow found (the `p=0` sup trajectory
`~ sqrt(K)`) has the wrong mechanism and dies at the physical vertex, so it is recorded as
a curiosity, not a bridge. **Next joint step, if any:** the only GU objects with a chance
at SAX typing are *counting*-side ones (mode counts under a concave capacity constraint,
e.g. horizon-area-type bounds), not cost-side ones -- a different swing, not this one.

**Adversary answers (self-critical).**

- *"You chose the domain to make it fail."* Four typings were run, including the
  maximally favorable trajectory steelman (T-D), which was **allowed** its best case
  (`p=0`, concave `sqrt(K)`) and honestly logged -- it still fails SAX-2's quantifier
  domain, SAX-3, and the physical vertex. The failure is over-determined, not
  domain-picked.
- *"Maybe additivity is an artifact of the model's mode-independence."* It is
  definitional, not modelic: any per-state quadratic form is a measure in its spectral
  variable; additivity over disjoint spectral sets survives any interaction that keeps the
  form well-defined. What interactions *could* add is correlation -- which would push the
  functional toward `P_corr` (entropy), still not SAX.
