# Explicit 2x2 Block Matrix Verification of E(xi)^{-1}

## 1. Problem Statement

The file `explorations/vz-evasion/vz-e-block-direct-clifford-2026-06-23.md` (§4) asserts the explicit
inverse of the E block:

```
E(xi)   = [[0,          (1/14)G  ],
           [(1/14)G,    (13/98)G ]]

E(xi)^{-1} = [[-(26/xi^2)G,  (14/xi^2)G],
              [ (14/xi^2)G,    0         ]]
```

where G = gamma(xi) in Cl(9,5) = M(64,H) satisfies G^2 = xi^2 * Id_S with xi^2 = g_Y(xi,xi) != 0.

The file included a partial in-line verification (positions (1,1), (1,2), (2,1), (2,2) of
E * E^{-1} only) but left OQ-CAS-E open: an independent explicit computation verifying both
E * E^{-1} = Id and E^{-1} * E = Id.

**Why this matters.** CORRECTION CR-04 (DERIVATION-PROGRESS.md) reverted the vz-14d-mixed-covectors
status from EVADED to CONDITIONALLY_EVADED, specifically because the E(xi) invertibility
argument was same-session reconstruction grade and the VZ-01 circularity flag cannot be closed
by a same-session file alone. The present file provides the independent explicit computation
that OQ-CAS-E requires.

**Scope.** This computation operates in the 2x2 block algebra over the ring of Clifford-algebra
endomorphisms of S = H^64. Each block entry is a scalar multiple of G (a degree-1 Clifford
element in Cl(9,5) = M(64,H)) or zero. The only algebraic fact used is G^2 = xi^2 * Id, which
is exact in any faithful representation of Cl(9,5) and in particular in M(64,H).

## 2. Setup and Notation

### 2.1 Block Decomposition

Q = E_0 + Im P_T where:
- E_0 = S^+ = H^32 (scalar spinor sector)
- Im P_T = {(1/14) gamma_A s : s in S^-} (gamma-trace sector of the one-form spinors)

The E block acts as:
```
E(xi): (phi, j(s)) in E_0 x Im P_T  |-->  ((1/14) G s,  (1/14) G phi + (13/98) G j(s))
```

In matrix form with ordered basis (E_0 entry, Im P_T entry):
```
E = [[E_{11}, E_{12}],  =  [[0,         A ],
     [E_{21}, E_{22}]]      [A,         B ]]
```

where:
- A = (1/14) G      (maps E_0 -> Im P_T and maps Im P_T -> E_0)
- B = (13/98) G     (maps Im P_T -> Im P_T)
- E_{11} = 0        (E_0 -> E_0)

### 2.2 Proposed Inverse

Write:
```
E^{-1} = [[X_{11}, X_{12}],  =  [[-26G/xi^2,  14G/xi^2],
          [X_{21}, X_{22}]]      [ 14G/xi^2,   0        ]]
```

### 2.3 Scalar Identities Used

All computations use only:

**(I1)** G^2 = xi^2 * Id_S    (Clifford identity, exact in Cl(9,5))

**(I2)** G^3 = G * G^2 = G * xi^2 * Id = xi^2 * G   (consequence of I1)

**(I3)** G^k = xi^{2(k-1)/2} * G for odd k, and G^k = xi^{k/2} * Id for even k.
         In particular: G * G = xi^2 Id, G * G * G = xi^2 G.

**(I4)** All entries in E and E^{-1} are scalar multiples of G or zero. Since G commutes
         with scalar multiples of itself, the product of any two entries is a scalar multiple
         of G^2 = xi^2 Id or G^1. Commutativity holds in the subalgebra generated by G.

## 3. Explicit Computation of E * E^{-1}

Compute M = E * E^{-1} entry by entry using standard 2x2 matrix multiplication:

```
M_{ij} = sum_k E_{ik} * X_{kj}
```

### Entry M_{11} = E_{11} * X_{11} + E_{12} * X_{21}

```
M_{11} = 0 * (-(26/xi^2) G) + A * (14/xi^2) G
       = 0 + (1/14) G * (14/xi^2) G
       = (14 / (14 xi^2)) * G^2
       = (1 / xi^2) * xi^2 * Id         [by (I1)]
       = Id
```

**Result: M_{11} = Id** ✓

### Entry M_{12} = E_{11} * X_{12} + E_{12} * X_{22}

```
M_{12} = 0 * (14/xi^2) G + A * 0
       = 0 + 0
       = 0
```

**Result: M_{12} = 0** ✓

### Entry M_{21} = E_{21} * X_{11} + E_{22} * X_{21}

```
M_{21} = A * (-(26/xi^2) G) + B * (14/xi^2) G
       = (1/14) G * (-(26/xi^2) G) + (13/98) G * (14/xi^2) G
```

**First term:**
```
(1/14) * (-(26/xi^2)) * G^2
= -(26 / (14 xi^2)) * xi^2         [by (I1)]
= -26/14
= -13/7
```

**Second term:**
```
(13/98) * (14/xi^2) * G^2
= (13 * 14 / (98 xi^2)) * xi^2     [by (I1)]
= 13 * 14 / 98
= 182 / 98
= 13/7
```

**Sum:**
```
M_{21} = -13/7 + 13/7 = 0
```

**Result: M_{21} = 0** ✓

### Entry M_{22} = E_{21} * X_{12} + E_{22} * X_{22}

```
M_{22} = A * (14/xi^2) G + B * 0
       = (1/14) G * (14/xi^2) G + 0
       = (14 / (14 xi^2)) * G^2
       = (1 / xi^2) * xi^2 * Id         [by (I1)]
       = Id
```

**Result: M_{22} = Id** ✓

### Summary: E * E^{-1} = Id (verified)

```
E * E^{-1} = [[Id, 0 ],
              [0,  Id]]
```

All four entries verified exactly. The only non-trivial check is M_{21}, where the
cancellation -13/7 + 13/7 = 0 is exact (not approximate). This cancellation is the
key arithmetic identity: it encodes the relationship between the (1/14)G off-diagonal
and (13/98)G diagonal entries of E.

## 4. Explicit Computation of E^{-1} * E

Compute N = E^{-1} * E entry by entry:

```
N_{ij} = sum_k X_{ik} * E_{kj}
```

### Entry N_{11} = X_{11} * E_{11} + X_{12} * E_{21}

```
N_{11} = (-(26/xi^2) G) * 0 + (14/xi^2) G * A
       = 0 + (14/xi^2) G * (1/14) G
       = (14 / (14 xi^2)) * G^2
       = (1 / xi^2) * xi^2 * Id         [by (I1)]
       = Id
```

**Result: N_{11} = Id** ✓

### Entry N_{12} = X_{11} * E_{12} + X_{12} * E_{22}

```
N_{12} = (-(26/xi^2) G) * A + (14/xi^2) G * B
       = (-(26/xi^2) G) * (1/14) G + (14/xi^2) G * (13/98) G
```

**First term:**
```
-(26/xi^2) * (1/14) * G^2
= -(26 / (14 xi^2)) * xi^2             [by (I1)]
= -26/14
= -13/7
```

**Second term:**
```
(14/xi^2) * (13/98) * G^2
= (14 * 13 / (98 xi^2)) * xi^2         [by (I1)]
= 182/98
= 13/7
```

**Sum:**
```
N_{12} = -13/7 + 13/7 = 0
```

**Result: N_{12} = 0** ✓

### Entry N_{21} = X_{21} * E_{11} + X_{22} * E_{21}

```
N_{21} = (14/xi^2) G * 0 + 0 * A
       = 0 + 0
       = 0
```

**Result: N_{21} = 0** ✓

### Entry N_{22} = X_{21} * E_{12} + X_{22} * E_{22}

```
N_{22} = (14/xi^2) G * A + 0 * B
       = (14/xi^2) G * (1/14) G + 0
       = (14 / (14 xi^2)) * G^2
       = (1 / xi^2) * xi^2 * Id         [by (I1)]
       = Id
```

**Result: N_{22} = Id** ✓

### Summary: E^{-1} * E = Id (verified)

```
E^{-1} * E = [[Id, 0 ],
              [0,  Id]]
```

All four entries verified exactly. The same cancellation -13/7 + 13/7 = 0 appears at
N_{12}, confirming the arithmetic is symmetric.

## 5. Structural Analysis of the Key Cancellation

The only non-trivial entry in both products is the (2,1) entry in E * E^{-1} and the
(1,2) entry in E^{-1} * E. Both reduce to the same cancellation:

```
-26/14 + 182/98 = -13/7 + 13/7 = 0
```

This can be written as a rational identity:

```
(1/14) * (26/1) = (13/98) * (14/1)
13/7 = 13/7
```

Or equivalently: the coefficients satisfy
```
A * X_{11} + B * X_{21} = 0
```
requires
```
(1/14) * (-26/xi^2) + (13/98) * (14/xi^2) = 0
-26/14 + 182/98 = 0
-182/98 + 182/98 = 0  [since 26/14 = 26*7/98 = 182/98]
```

This is exact integer arithmetic: 26 * 7 = 182 and 13 * 14 = 182. The cancellation is
an exact consequence of the specific coefficients (1/14) and (13/98) in the E matrix.

These coefficients arise from the 14D Clifford contraction identity:
```
gamma^A gamma_A = 14 Id       (trace in 14D)
gamma^A gamma(xi) gamma_A = -12 gamma(xi)    (contracted Clifford element, 14D)
```

The coefficient 13/98 = 13/(7*14) encodes the ratio (14-2)/14 = 12/14 = 6/7 from
Clifford contraction in D=14, while 1/14 comes from the RS projector normalization.
The inverse coefficients -(26/xi^2) and (14/xi^2) are uniquely determined by these.

## 6. Commutativity Check

The computation above treats G as an element of the center of the subalgebra {scalar * G, scalar * Id}.
This is valid because:

1. G = gamma(xi) is a fixed Clifford element (fixed covector xi).
2. All entries in E and E^{-1} are scalar multiples of G or zero.
3. Products of such entries are scalar multiples of G^2 = xi^2 Id or G, both of which
   commute with everything in this subalgebra.

Therefore the order of multiplication does not matter within this subalgebra, and both
E * E^{-1} and E^{-1} * E give the same result (Id), which is consistent.

**Non-commutativity concern:** In the full Clifford algebra Cl(9,5), distinct Clifford
elements do not commute. However, this computation involves only a single Clifford element
G (fixed xi) and scalar coefficients. The subalgebra generated by {Id, G} is commutative
since G * G = G^2 = xi^2 Id (a scalar). All matrix products here stay within this
commutative subalgebra. This is not an approximation; it is exact.

## 9. Explicit Failure Conditions

The RESOLVED verdict for the E^{-1} formula would be falsified by:

**F1.** The E matrix formula E = [[0, (1/14)G], [(1/14)G, (13/98)G]] being wrong. The
present computation takes this as input (verified from the Clifford module structure of
sigma_D in vz-e-block-direct-clifford §2). If the E matrix has additional entries or
different coefficients, the inverse formula changes. This is a question about the GU
operator structure, not about the present matrix computation.

**F2.** G^2 != xi^2 * Id. This would falsify every step where G^2 is substituted. This
identity is exact in Cl(9,5) = M(64,H) for any covector xi; it is the defining relation
of the Clifford algebra. It cannot fail in any faithful representation.

**F3.** An arithmetic error in the computation. The key integer identity is 26 * 7 = 182
and 13 * 14 = 182, giving -26/14 + 182/98 = -182/98 + 182/98 = 0. This is elementary
integer arithmetic and is correct.

No other failure conditions are possible: given the E matrix and G^2 = xi^2 Id, the
computation is a finite sequence of exact rational arithmetic steps with no approximations.
