## What Is Genuinely Algebraic

The following pieces are algebraic once the stated block model is accepted:

1. `G^2 = xi2 Id` implies `G` is invertible whenever `xi2 != 0`, with
   `G^{-1} = G / xi2`.
2. For the displayed E-block, the direct kernel proof is clean:
   if `E(phi, j(s)) = 0`, the first row gives `G s = 0`, hence
   `xi2 s = 0`, hence `s = 0`; the second row then gives `G phi = 0`,
   hence `phi = 0`.
3. The proposed inverse

```text
E(xi)^(-1) = [[-(26/xi2) G, (14/xi2) G],
              [ (14/xi2) G, 0          ]]
```

   multiplies to the identity on both sides in the quotient algebra generated by
   `G` with relation `G^2 = xi2`.
4. If E is independently established invertible, the Schur kernel argument is also
   algebraic: for `S_R psi_R = 0`, set `v = -E^{-1} C psi_R`; then
   `M(xi)(psi_R, v) = 0`, so `M(xi)^2(psi_R, v) = xi2 (psi_R, v) = 0`.
   For `xi2 != 0`, this forces `psi_R = 0`.

These are not the circular determinant argument. They do not use
`det(M) = det(E) det(S_R)` to infer E-invertibility.

## Proof-Grade Verification Contract

### E-block invertibility contract

A proof-grade closure of FC-VZ-1 should provide, in a later tracked/reviewed pass or by
external verification:

1. A definition-level derivation of the Q-sector splitting used in the computation:
   `Q = E_0 direct-sum Im P_T`, including dimensions and projectors.
2. A derivation of

```text
E(xi) = P_Q sigma_D(xi) P_Q | Q
      = [[0,        (1/14) gamma(xi)],
         [(1/14) gamma(xi), (13/98) gamma(xi)]]
```

   from the Clifford/gamma-trace definitions, with sign, signature, chirality, and
   H-linearity conventions fixed.
3. A proof, independent of Schur/determinant factorization, that `gamma(xi)^2 =
   g_Y(xi,xi) Id` in the same convention and for all real `xi in T*Y^14`.
4. A two-sided inverse or kernel proof for all `xi2 != 0`, performed over a faithful
   real/quaternionic representation or a formal Clifford quotient, not just at sample
   covectors.
5. A null-locus check showing E fails only at `xi2 = 0`, and that no non-null mixed
   horizontal/vertical covector creates a Q-sector kernel.
6. A clear provenance statement: the check is inter-session or external to the
   2026-06-23 VZ-01 / CR-04 loop.

Representative numerical 64x64 tests would be useful smoke tests, but proof-grade
closure should be symbolic or theorem-backed; random examples alone cannot quantify over
all mixed covectors.

### Schur complement contract

Once E is proof-grade invertible, the Schur step must be recorded without determinant
circularity:

1. Define the same block decomposition `M = [[A, B], [C, E]]` and verify that `E^{-1} C`
   is a well-defined Q-sector operator for every non-null `xi`.
2. Define `S_R = A - B E^{-1} C` only after E-invertibility is established.
3. Prove the kernel implication directly:
   `S_R psi_R = 0 -> M(psi_R, -E^{-1} C psi_R) = 0 -> xi2(psi_R, ...) = 0 -> psi_R = 0`.
4. If determinants/characteristic polynomials are mentioned, translate them through a
   real representation and keep them downstream of the kernel proof.
5. Explicitly state scope: this closes the 14D principal-symbol no-spacelike-kernel claim,
   not the full lower-order/dynamical problem.

This confirms the inverse arithmetic in the simplified block algebra. It does not verify
the actual `Cl(9,5) = M(64,H)` matrix model, the derivation of the coefficients, or the
Q-sector projection. Treat it as a sanity check only.
