# Adapter-assumed four-leg swing

what does each leg need the adapter to TRANSMIT — informed by what
possibility-to-capability demonstrated today — and is there a possible
construction where all four legs fit?

The p2c lesson used throughout: a Z2-valued global datum can be stored
topologically — locally invisible (no local probe reads it), globally

mimic-rejecting controls. This is a concrete existence proof for the KIND of
datum GU keeps needing: external, global, finite, not locally forgeable.

indefinite Krein space is physical (the bar-b bit; tri-repo result: one
external Z/2 posit, home p2c). p2c just showed a Z2 sector datum can live
topologically.

**Hypothesis:** the adapter transmits a single global Z/2 orientation — a
topological sector datum, not a local field. That orientation fixes the
Krein sign, defines the source-side differential `d_RS,-1`, and the six P4
certificates follow from the graded quotient it induces.

**Adapter payload:** 1 global Z/2, loop-coherent (locally unreadable, so
the choice cannot be retuned target-locally — the non-retuning condition is
structural, not procedural).

**First falsification test:** build the finite graded-quotient toy (mirror
p2c's fixture style): does grading the Krein toy by an external Z/2 produce
a positive-definite physical sector with state-preserving dynamics — and
does the WRONG orientation visibly fail (negative-norm survivor)? If both
orientations "work," the hypothesis is empty and dies.
