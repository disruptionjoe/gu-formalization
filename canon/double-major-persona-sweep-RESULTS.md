# What Are We Missing: Double-Major Persona Sweep -> Hegelian -> Meta-Synthesis

## 1. The one thing that matters

Stripped of ornament, the entire double-major sweep terminates at a single instruction that was already on
the table: compute the one projected/twisted Rarita-Schwinger index on GU's actual 14-manifold, with the
tangential-vs-gauge fork (e = 1/12 vs e = 3/8) carried as an explicit case split fixed by normalizability plus
reflection positivity. Every grounded lens, however exotic its second major, re-locates the same empty slot
rather than filling it. What the sweep changed is not the verdict but the *standing* of two things around it: it
hardened the canon's tentative "possible category error" flag into a near-theorem (order-3-class -> integer-3 is
ill-typed because Hom(Z/3, Z) = 0, so no canonical class-to-count map exists), and it produced a small number
of genuinely new, runnable probes (a Hessian eigenvalue along the Z/3 direction, a percolation-style robustness
test of the exact-0 decouple, and an IR anomaly-matching check for e_R = 1/12). "Located, not forced" survives
intact; the open conjecture is now sharper and better-typed.

## 2. Convergent themes, ranked and graded

bears directly on the computed math: the forcing-slot test already showed every produced integer is 2-primary
or the Z/3 identity, and this theme explains *why* structurally -- there is no canonical map for an integer to
live in. Upgrades the canon flag from suspected to provable.

geometry-dependent. [GROUNDED/TESTABLE]**
~10 personas (k-invariant of the gauge-tangent pair, Z/3-equivariant Lefschetz number, K-theory/holographic-code

relocation is not an escape from the named computation, it is independent motivation for it from many sides.
This is the constructive complement to theme 1: the absolute class cannot be an integer, but a relative/rank
object can, and that is exactly what the single decider is.

**3. Characterize the bridge as a concrete object, not a named gap. [GROUNDED/TESTABLE]**

**5. The single integer computation is the unique load-bearing next step. [GROUNDED]**
The terminal meta-convergence: every additional lens, double-majored or not, re-locates the same empty slot

**6. The tangential-vs-gauge fork is a frame/boundary-condition choice the count is a function of, not a flaw.

Fixed by normalizability plus reflection positivity (weighted-Fredholm/APS, Cartan, GR, dimensional

**Quarantine (FANCIFUL/unanchored):** any reading that outputs a *cardinal* (dies by the 2-primary lemma); any

## 3. The Hegelian sublations that matter, brought back to the math

**Derived vs Selected -> the index computation IS the search for an attractor.** The forcing-slot test built the
gated object four ways and never reached all three of {tangential, net-chiral, non-frame-trivial} with a
3-primary integer -- consistent with both "fixed integer not yet read off" and "occupant set dynamically." The
sublation makes this decidable: compute the twisted-RS index *and* the Hessian eigenvalue of the GU action along
the Z/3 direction. Eigenvalue exactly 0 means the count is a genuine modulus (a flat direction / zero mode), so
"located not forced" is true at the dynamical level, not merely arithmetically; nonzero-stable landing on a
tangentially-framed integer means forced. The single sharpest upgrade the sweep produced -- a static-topology
sweep could not have posed it.

**Missing computation vs type error -> relocate to a rank home.** The absolute class e_R = 1/12 is homotopy-fixed
and cannot co-vary; theme 1 says it therefore cannot *be* the count. Build the count instead in a relative/
equivariant/rank group that admits Z by construction. The twisted-RS index is already such an object (a Fredholm
index is integer-valued by definition); the work is to write it on the actual 14-manifold with the gauge-tangent
pair as the relative datum.

**In-an-arena vs on-the-bridge -> compute the one edge.** Every single-arena lens collapses to one empty slot;
that collapse is the *signature* of node-only inspection. Pontryagin duality supplies the discipline: the
linking pairing between coprime cyclics Z/8 x Z/3 -> Q/Z vanishes identically, so the bridge cannot be a
canonical pairing -- precisely why it resisted construction and why node-inspection found nothing. Characterize
the inflow as a linking form / Cech H^1 cocycle / graph b_1 and check its output alphabet; a non-canonical lift
carrying odd torsion across would be the first real motion of LOCATE toward FORCE.

**Convergence vs illusion -> universality-class meta-datum, MDL headline.** 45 lenses flowing to one fixed point
is a d-connected single datum about the answer but an informative datum about the *question* (an RG universality
class whose basin dimension counts relevant directions). Price the claim by MDL: "located + one carrier + a
selector" is minimum-description-length-optimal and the honest headline; "forced" is strictly longer,
currently-unpaid description.

**Carrier-side vs selector-side forcing -> the honest forceable statement is a congruence or inequality.** The
count was located in the odd-torsion carrier (e_R = 1/12), but the only physically-attested "3" -- Kobayashi-
Maskawa needing n >= 3 for a complex CP-violating phase -- lives in the 2-primary, T-odd selector arena, on the
wrong side of the firewall. There may be two distinct threes. A defensible smaller theorem is type-correct where
"the carrier integer equals 3" is not: prove "net chiral count != 0 mod 3" (carrier-side congruence) or
"n >= 3" (selector-side extremal threshold). Either is falsifiable now and adds the missing fourth bin --
extremal-threshold -- to the cardinal / 2-primary-class / order-3-carrier sort.
