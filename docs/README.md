# docs/

Second-tier program documents -- context and orientation that an outsider does not need at the front door,
but that fills in the picture. The primary entry points stay at the repo root (`README.md`,
`RESEARCH-PROGRAM.md`, `RESEARCH-POSTURE.md`, `CANON.md`) along with the live status docs
(`RESEARCH-STATUS.md`, `NEXT-STEPS.md`, `DERIVATION-PROGRESS.md`).

- [`OVERVIEW.md`](OVERVIEW.md) — high-level overview of the program.
- [`NEXT-FRONTIER-HYPOTHESES.md`](NEXT-FRONTIER-HYPOTHESES.md) — candidate hypotheses for the frontier
  (what lies outside the observer universe).
- [`WHERE-GU-STANDS-AND-THE-MISSING-OBJECT-2026-06-27.md`](WHERE-GU-STANDS-AND-THE-MISSING-OBJECT-2026-06-27.md)
  — dated deep-dive on the state of the GU reconstruction and the missing matter action.
- [`paper-formalization-candidates.md`](paper-formalization-candidates.md) — inventory of results that could
  become papers.
