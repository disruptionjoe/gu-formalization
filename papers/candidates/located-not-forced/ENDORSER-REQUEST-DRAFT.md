## How the mechanics work (so the note makes sense)

1. Start the submission on arXiv first and pick `hep-th`. arXiv emails **you** an endorsement request
   containing a **6-character code** and a link.
2. You then contact **one** eligible endorser and give them that code. They endorse at
   `arxiv.org/auth/endorse` (or via the link). One positive endorsement clears the category.
3. Only after that can you complete the submission.

So the code in the note below comes from arXiv *after* you begin -- fill it in then.
