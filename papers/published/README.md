# papers/published/

The **public record** -- only papers that have actually been **published publicly** (arXiv, a journal, a live
blog post). **Empty by default.** Joe moves a paper up here by hand when, and only when, it actually goes
public, recording the venue / date / arXiv id.

Near-final papers that Joe has decided to publish but has not yet posted live in the sibling
[`../candidates/`](../candidates/). The full three-stage lifecycle (`drafts/` -> `candidates/` -> `published/`)
is described in [`../README.md`](../README.md).

Treat as append-mostly: do not silently rewrite a published artifact -- publish a revision and note it.
